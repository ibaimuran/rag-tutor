## 第3章 高级提示词策略

### 3.1 提示词组合与模板化

#### 1. 核心概念

想象一下，你是一位大厨，需要为不同的客人定制菜单。如果每次都要从头开始思考“放多少盐、用什么火候”，不仅效率低下，而且容易出错。聪明的做法是，先准备好几个“万能菜谱框架”，比如“清蒸鱼框架”、“红烧肉框架”。每次做菜时，你只需要在这个框架里填入具体的食材（比如“鲈鱼”或“草鱼”）和微调口味（比如“多放点姜”），就能快速、稳定地做出一道好菜。

**提示词组合与模板化**，就是为AI（如ChatGPT）准备这种“万能菜谱框架”。它指的是将多个基础提示词技巧（如角色扮演、格式指定、任务分解）有机地组合成一个结构化的、可复用的**模板**。这样，当你需要AI完成一系列相似任务时，就不必每次都写一大段全新的提示词，而是直接套用模板，填入关键信息即可。这极大地提高了与AI协作的效率和输出质量的一致性。

#### 2. 关键定义

-   **提示词组合 (Prompt Composition)**：将两个或以上的基础提示词策略（例如：**角色扮演** + **分步指令** + **输出格式控制**）融合在一个提示词中，以实现更复杂、更精准的指令。它不是简单堆砌，而是有机整合。
-   **提示词模板 (Prompt Template)**：一个预先设计好的、结构化的提示词框架。它包含固定不变的**指令骨架**和需要用户填写的**变量占位符**（通常用`{变量名}`表示）。模板化是提示词组合的高级应用形式。
-   **变量占位符 (Variable Placeholder)**：在模板中，用特定符号（如`{ }`）标记出来的位置，用于在每次使用时填入不同的具体信息，例如`{主题}`、`{目标受众}`、`{字数要求}`等。
-   **可复用性 (Reusability)**：模板的核心价值。一个好的模板可以反复用于处理同一类任务，只需改变变量值，就能获得不同但结构相似的输出。

#### 3. 示例与应用

**例子1：撰写产品介绍邮件**

-   **基础提示词**：“帮我写一封邮件，介绍我们的新产品。”
-   **组合模板化后的提示词**：

    ```
    # 角色
    你是一位专业的营销文案专家。

    # 任务
    请根据以下信息，撰写一封用于推广新产品的营销邮件。

    # 输入信息
    - 产品名称：{产品名称}
    - 核心卖点：{卖点1}、{卖点2}、{卖点3}
    - 目标客户：{目标客户群体}
    - 邮件目的：{例如：邀请试用 / 限时折扣}

    # 输出要求
    1. 邮件主题行要吸引人点击。
    2. 正文开头要引起共鸣，中间突出产品价值，结尾要有明确的行动号召（CTA）。
    3. 语气专业且热情。
    4. 全文不超过200字。
    ```

    **应用**：下次要推广“智能扫地机器人”时，只需将`{产品名称}`替换为“智能扫地机器人”，`{卖点1}`替换为“激光导航”，`{目标客户}`替换为“忙碌的上班族”等，就能快速生成一封高质量的营销邮件。

**例子2：制定学习计划**

-   **基础提示词**：“帮我制定一个学习计划。”
-   **组合模板化后的提示词**：

    ```
    你是一位资深的教育规划师。请为我制定一个为期{学习周期}的{学习主题}学习计划。

    我的当前水平是：{例如：零基础 / 有基础但需要巩固}。
    我每周可以投入的学习时间是：{小时数}小时。
    我的学习目标是：{例如：通过考试 / 掌握实操技能}。

    请按周分解计划，每周包含：
    1. 本周学习重点。
    2. 推荐的学习资源（书籍、视频、网站）。
    3. 需要完成的实践练习。
    ```

    **应用**：想学Python，就把`{学习主题}`换成“Python编程”，`{学习周期}`换成“8周”，`{小时数}`换成“10”，就能得到一个量身定制的学习路线图。

#### 4. 常见误区

-   **误区一：模板就是“复制粘贴”**
    -   **错误理解**：认为模板是万能的，直接套用就能得到完美答案，忽略了根据具体场景调整模板的必要性。
    -   **正确认知**：模板是“骨架”，需要你根据每次任务的细微差别，灵活调整模板中的指令或增加新的约束条件。例如，上面的产品邮件模板，如果客户是“Z世代年轻人”，你可能需要在“语气”部分加上“使用网络流行语”。

-   **误区二：模板越复杂越好**
    -   **错误理解**：认为模板里包含的指令越多、越详细，AI表现就越好。
    -   **正确认知**：过于冗长和复杂的模板可能会让AI“迷失重点”，反而降低输出质量。好的模板应该**结构清晰、指令明确、重点突出**。遵循“奥卡姆剃刀”原则，在保证效果的前提下，模板应尽可能简洁。

#### 5. 思考题

假设你是一位老师，需要每周为不同班级（如“基础班”和“提高班”）的学生生成一份包含“复习要点”、“新知识讲解”和“课后作业”的周学习总结。请你尝试设计一个通用的提示词模板，并说明这个模板如何通过改变变量值来适应两个不同班级的需求。
