# 第三单元 物质构成的奥秘

## 课题2 原子结构

### 1. 核心概念

同学们，我们已经知道，物质是由分子、原子等微观粒子构成的。那么，原子本身又是怎样的呢？它还能不能再分？

想象一下，原子就像一个微缩的“太阳系”。在它的中心，有一个非常小、非常致密的“原子核”（nucleus），就像太阳一样。原子核虽然小，但它几乎集中了原子的全部质量。在原子核的周围，有一些更小的、带负电的粒子——**电子**（electron），它们就像行星一样，在高速运动着，形成一层“电子云”。

所以，原子的结构可以简单地概括为：**一个位于中心的、带正电的原子核，和核外若干个带负电的电子**。原子核本身也不是“实心”的，它还可以再分为带正电的**质子**（proton）和不带电的**中子**（neutron）。

### 2. 关键定义

*   **原子**：化学变化中的最小粒子。在化学变化中，原子核不变，只是核外电子发生重新组合。
*   **原子核**：位于原子中心，由**质子**和**中子**构成，带**正电荷**。
*   **质子**：原子核中带**正电荷**的粒子。一个质子带一个单位的正电荷。**质子数**决定了原子的种类（例如，所有氢原子都有1个质子）。
*   **中子**：原子核中**不带电**的粒子。它的主要作用是“粘合”质子，使原子核稳定。
*   **电子**：原子核外带**负电荷**的粒子。一个电子带一个单位的负电荷。电子的质量极小，约为质子质量的1/1836。
*   **核电荷数**：原子核所带的正电荷数，它等于**质子数**。
*   **原子序数**：元素周期表中给元素排的序号，它等于**质子数**。
*   **电中性**：在原子中，**质子数 = 核外电子数**，所以正负电荷总量相等，整个原子不显电性。

### 3. 示例与应用

1.  **为什么铁会生锈？**
    铁生锈是铁原子与空气中的氧气分子发生化学反应。在这个反应中，铁原子失去了一些核外电子，变成了带正电的**铁离子**（Fe²⁺或Fe³⁺），并与氧离子结合形成铁锈（主要成分是氧化铁）。这个过程就是原子核外电子的“搬家”导致的。

2.  **为什么氢气球能飞起来？**
    氢气（H₂）分子由两个氢原子构成。每个氢原子的原子核里只有1个质子，没有中子，核外有1个电子。因为氢原子是所有原子中质量最轻的，所以由它组成的氢气密度比空气小得多，因此氢气球能飘起来。

3.  **为什么钻石（金刚石）和铅笔芯（石墨）都是碳，但性质完全不同？**
    它们都是由碳原子构成的。区别在于，碳原子在金刚石和石墨中的**排列方式**（即原子间的空间结构）不同。金刚石中的碳原子排列成紧密的立体网状结构，非常坚硬；而石墨中的碳原子排列成层状结构，层与层之间容易滑动，所以石墨很软，能用来写字。这告诉我们，原子的“堆叠方式”决定了物质的宏观性质。

### 4. 常见误区

*   **误区一：原子是实心的小球。**
    **纠正**：原子内部绝大部分是“空”的。原子核的直径只有原子直径的几万分之一到十万分之一。如果把一个原子放大到一个足球场那么大，原子核就相当于场地中央的一颗绿豆。电子就在这个巨大的空间里高速运动。

*   **误区二：原子核内的质子数和中子数总是相等的。**
    **纠正**：对于大多数原子来说，中子数约等于或略大于质子数，但并不总是相等。例如，氢原子（¹H）有1个质子，0个中子；而另一种氢原子（氘，²H）有1个质子，1个中子。同种元素的不同原子，其质子数相同，但中子数可以不同，我们称它们为**同位素**（isotope）。

### 5. 思考题

**思考题：**
我们已经知道，原子核带正电，电子带负电。根据“异性相吸”的原理，电子应该会被原子核吸引过去，最终撞到原子核上。但为什么原子中的电子没有掉进原子核里，而是能稳定地绕核运动呢？这背后隐藏着怎样的科学原理？
