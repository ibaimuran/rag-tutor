# Python编程入门

- 学科：Computer Science
- 年级：7-9年级
- 章节数：10

## 第1单元 Python初探

- **认识Python**（难度：★，前置依赖：无）
  介绍Python语言的历史、特点和应用领域，安装Python开发环境，学习使用IDLE或命令行运行第一个Python程序。
- **第一个Python程序**（难度：★，前置依赖：认识Python）
  学习编写和运行简单的Python程序，掌握print()函数输出文本，理解程序的基本结构。

## 第2单元 变量与数据类型

- **变量与赋值**（难度：★★，前置依赖：第一个Python程序）
  理解变量的概念，学习如何创建变量并赋值，掌握变量命名规则和常见错误。
- **基本数据类型**（难度：★★，前置依赖：变量与赋值）
  学习整数、浮点数、字符串和布尔值四种基本数据类型，掌握类型转换函数int()、float()、str()。
- **输入与输出**（难度：★★，前置依赖：基本数据类型）
  学习使用input()函数获取用户输入，结合print()输出结果，实现简单的交互程序。

## 第3单元 运算符与表达式

- **算术运算符**（难度：★★，前置依赖：基本数据类型）
  学习加、减、乘、除、整除、取余和幂运算等算术运算符，掌握运算符优先级和表达式求值。
- **比较与逻辑运算符**（难度：★★★，前置依赖：算术运算符）
  学习比较运算符（==、!=、<、>等）和逻辑运算符（and、or、not），用于条件判断。

## 第4单元 控制流程

- **条件语句**（难度：★★★，前置依赖：比较与逻辑运算符）
  学习if、elif、else语句实现分支结构，掌握条件表达式的写法，解决简单判断问题。
- **循环语句**（难度：★★★，前置依赖：条件语句）
  学习for循环和while循环，掌握range()函数，理解循环嵌套和break、continue语句。

## 第5单元 字符串操作

- **字符串索引与切片**（难度：★★★，前置依赖：基本数据类型）
  学习字符串的索引（正索引和负索引）和切片操作，提取子字符串。
- **字符串方法**（难度：★★★，前置依赖：字符串索引与切片）
  学习常用字符串方法如upper()、lower()、strip()、split()、join()、replace()等，进行文本处理。

## 第6单元 列表与元组

- **列表的创建与操作**（难度：★★★，前置依赖：循环语句）
  学习创建列表，使用索引和切片访问元素，掌握append()、insert()、remove()、sort()等方法。
- **元组与列表的区别**（难度：★★★，前置依赖：列表的创建与操作）
  学习元组的创建和不可变性，理解元组与列表的区别，掌握元组的应用场景。

## 第7单元 字典与集合

- **字典的创建与操作**（难度：★★★★，前置依赖：列表与元组）
  学习创建字典，使用键访问值，掌握keys()、values()、items()、update()等方法。
- **集合的创建与操作**（难度：★★★★，前置依赖：字典的创建与操作）
  学习创建集合，掌握add()、remove()、union()、intersection()等集合运算。

## 第8单元 函数

- **函数的定义与调用**（难度：★★★★，前置依赖：控制流程）
  学习使用def定义函数，理解参数（位置参数、默认参数、关键字参数）和返回值。
- **变量的作用域**（难度：★★★★，前置依赖：函数的定义与调用）
  理解局部变量和全局变量的区别，学习global关键字，掌握作用域规则。

## 第9单元 文件操作

- **文件的读写**（难度：★★★★，前置依赖：函数）
  学习使用open()打开文件，read()、write()、readlines()等方法读写文本文件，掌握with语句。
- **异常处理**（难度：★★★★，前置依赖：文件的读写）
  学习使用try-except语句捕获和处理异常，理解常见异常类型，提高程序健壮性。

## 第10单元 综合项目

- **项目：简易计算器**（难度：★★★★★，前置依赖：异常处理）
  综合运用变量、运算符、条件语句和循环，编写一个支持加、减、乘、除的交互式计算器程序。
- **项目：学生成绩管理系统**（难度：★★★★★，前置依赖：项目：简易计算器）
  综合运用列表、字典、函数和文件操作，实现一个能添加、删除、查询和保存学生成绩的管理系统。
