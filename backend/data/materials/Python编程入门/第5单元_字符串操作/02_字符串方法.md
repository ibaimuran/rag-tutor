## 第5单元 字符串操作

### 知识点：字符串方法

#### 1. 核心概念

想象一下，你有一串珍珠项链（这就是一个**字符串**）。字符串方法，就是一系列专门用来“打理”这串项链的“小工具”。你可以用它们来：

-   **检查**：看看项链上有没有某个特定颜色的珍珠（比如，是否包含字母“a”）。
-   **整理**：把所有珍珠都换成大写字母（比如，全部变成大写）。
-   **拆分**：把项链从某个连接处断开，变成几段更短的项链（比如，按空格拆成单词）。
-   **替换**：把项链上某些坏掉的珍珠换成新的（比如，把“坏”字替换成“好”字）。

在Python中，**字符串方法**就是**属于字符串**的特殊函数。你只需要在字符串变量后面加上一个点`.`，然后写上方法的名字和括号`()`，就能调用这些强大的功能。例如：`my_string.upper()`。

#### 2. 关键定义

-   **字符串（String）**：由零个或多个字符组成的**不可变**序列。在Python中，用单引号`'...'`或双引号`"..."`括起来。
-   **方法（Method）**：与特定数据类型（如字符串）相关联的函数。它通过点号`.`来调用。
-   **返回值（Return Value）**：方法执行后，会返回一个新的结果。**注意**：字符串方法**不会**修改原来的字符串（因为字符串是不可变的），而是返回一个处理后的**新字符串**。
-   **链式调用（Chaining）**：可以连续使用多个方法，例如 `my_string.strip().upper()`，表示先去除空格，再转换为大写。

#### 3. 示例与应用

**生活例子1：用户输入清洗**
用户在网站上注册时，经常不小心在名字前后输入了空格，比如`" 张三 "`。我们可以用`strip()`方法去除首尾空格，再用`title()`方法将名字格式化为首字母大写。

```python
user_input = "   zhangsan   "
cleaned_name = user_input.strip()  # 结果是 "zhangsan"
formatted_name = cleaned_name.title() # 结果是 "Zhangsan"
print(f"欢迎您，{formatted_name}！") # 输出：欢迎您，Zhangsan！
```

**生活例子2：数据验证**
检查用户输入的邮箱地址是否包含`@`符号，这是判断邮箱格式是否有效的基本步骤。

```python
email = "example@python.org"
if email.find("@") != -1:  # find()方法找不到会返回-1
    print("邮箱格式有效")
else:
    print("邮箱格式无效，缺少@符号")
```

**生活例子3：文本分析**
统计一段文字中某个单词出现的次数。

```python
sentence = "Python is fun. Python is powerful. I love Python."
count = sentence.count("Python")
print(f"“Python”这个词出现了 {count} 次。") # 输出：“Python”这个词出现了 3 次。
```

#### 4. 常见误区

-   **误区一：认为方法会修改原字符串**
    -   **错误理解**：`name = "Alice"`，执行`name.upper()`后，认为`name`的值变成了`"ALICE"`。
    -   **正确理解**：`name.upper()`会返回一个新字符串`"ALICE"`，但`name`本身仍然是`"Alice"`。你需要用一个新变量来接收返回值，例如`new_name = name.upper()`。

-   **误区二：混淆`find()`和`index()`方法**
    -   **错误理解**：认为两者完全一样。
    -   **正确理解**：两者都用于查找子字符串的位置。区别在于：如果找不到，`find()`会返回`-1`，而`index()`会直接**报错**（抛出`ValueError`异常）。在不确定子字符串是否存在时，使用`find()`更安全。
