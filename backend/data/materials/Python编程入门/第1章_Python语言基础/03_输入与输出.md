## 第1章 Python语言基础

### 1.3 输入与输出

#### 1. 核心概念

想象一下，你正在和一台聪明的计算机对话。为了让计算机帮你解决问题，你需要告诉它一些信息（比如“帮我计算1+1等于几？”），这就是**输入**。计算机处理完毕后，会把结果告诉你（比如“结果是2”），这就是**输出**。

在Python编程中，**输入**是指程序从外部世界（比如键盘、文件、网络）获取数据的过程。**输出**则是程序将处理结果展示给外部世界（比如屏幕、文件、打印机）的过程。最基础、最常用的输入和输出方式，就是通过键盘输入和屏幕显示。

#### 2. 关键定义

-   **输出函数 `print()`**：这是Python中最常用的输出工具。它的作用是将括号内的内容显示在屏幕上。你可以输出文字、数字，甚至是多个内容组合在一起。
    -   **语法**：`print(要输出的内容)`
    -   **示例**：`print("Hello, World!")` 会在屏幕上显示 `Hello, World!`。
-   **输入函数 `input()`**：这是Python中获取用户键盘输入的工具。当程序执行到 `input()` 时，会暂停等待用户输入。用户输入完内容并按下回车键后，`input()` 函数会**将用户输入的内容作为一个字符串（string）返回**。
    -   **语法**：`变量 = input("提示信息")`
    -   **示例**：`name = input("请输入你的名字：")` 程序会先显示“请输入你的名字：”，等待用户输入，用户输入“小明”并回车后，变量 `name` 的值就变成了字符串 `"小明"`。

#### 3. 示例与应用

1.  **生活例子：自动售货机**
    -   **输入**：你投入硬币（输入金额）并按下商品按钮（输入商品编号）。
    -   **处理**：售货机内部计算金额是否足够，并找到对应商品。
    -   **输出**：售货机弹出商品并显示“找零XX元”（输出结果）。
    -   **Python模拟**：
        ```python
        # 输入：用户选择商品
        choice = input("请选择商品编号（1.可乐 2.雪碧）：")
        # 输出：显示用户选择
        print("你选择了商品编号：" + choice)
        ```

2.  **应用场景：简单的计算器**
    -   程序询问用户两个数字，然后计算它们的和并显示结果。
    -   **注意**：`input()` 返回的是字符串，不能直接用于数学计算，需要先用 `int()` 或 `float()` 函数将其转换为数字。
    ```python
    # 输入两个数字
    num1 = input("请输入第一个数字：")
    num2 = input("请输入第二个数字：")

    # 将字符串转换为整数
    num1_int = int(num1)
    num2_int = int(num2)

    # 计算和
    result = num1_int + num2_int

    # 输出结果
    print("两个数字的和是：", result)
    ```

3.  **应用场景：个性化问候**
    -   程序询问用户的名字，然后根据名字发送一条问候信息。
    ```python
    # 输入：获取用户姓名
    user_name = input("你好！请问你叫什么名字？")
    # 输出：个性化问候
    print("欢迎你，" + user_name + "！祝你学习愉快。")
    ```

#### 4. 常见误区

-   **误区一：忘记 `input()` 返回的是字符串**
    -   **错误理解**：认为 `input()` 返回的数字可以直接用于计算。
    -   **典型错误**：
        ```python
        age = input("请输入你的年龄：")
        next_year_age = age + 1  # 错误！age是字符串，不能和数字相加
        print("明年你", next_year_age, "岁。")
        ```
    -   **正确做法**：使用 `int()` 或 `float()` 进行类型转换。
        ```python
        age = int(input("请输入你的年龄："))
        next_year_age = age + 1
        print("明年你", next_year_age, "岁。")
        ```

-   **误区二：混淆 `print()` 中逗号 `,` 和加号 `+` 的作用**
    -   **错误理解**：认为逗号和加号在 `print()` 中作用相同。
    -   **区别**：
        -   **逗号 `,`**：可以连接不同类型的多个内容，并在输出时自动添加一个空格。例如 `print("年龄", 18)` 输出 `年龄 18`。
        -   **加号 `+`**：只能连接两个字符串，用于字符串拼接。如果连接数字和字符串会报错。例如 `print("年龄" + 18)` 会报错，因为 `18` 是数字不是字符串。
