## 第6单元 函数与模块

### 6.2 参数与作用域

#### 1. 核心概念

想象一下，你有一个功能强大的“榨汁机”（函数）。每次使用它，你都需要放入不同的水果（**参数**），才能得到不同的果汁（**返回值**）。参数就是函数从外部接收信息的“入口”，让同一个函数能处理不同的数据，变得非常灵活。

而“作用域”则像一个“工作区域”。在厨房里，你只能使用厨房里的工具和食材，不能直接去卧室拿东西。在Python中，每个变量都有自己生效的“工作区域”，这个区域就是它的作用域。函数内部定义的变量，通常只能在函数内部使用，就像厨房里的专用工具；而函数外部定义的变量，则可以在更大的范围内使用，就像客厅里共用的物品。

理解参数和作用域，是掌握函数如何“接收信息”和“隔离信息”的关键。

#### 2. 关键定义

- **参数 (Parameter)**：在**定义函数**时，写在函数名后面括号里的变量。它像一个“占位符”，用来接收将来调用函数时传入的数据。例如：`def greet(name):` 中的 `name` 就是一个参数。
- **实参 (Argument)**：在**调用函数**时，实际传递给函数的值。它会被赋值给对应的参数。例如：`greet("小明")` 中的 `"小明"` 就是一个实参。
- **局部作用域 (Local Scope)**：在**函数内部**创建的变量所处的区域。这些变量只能在函数内部被访问和修改，函数执行完毕后，它们通常会被销毁。这保证了函数内部的数据不会意外地影响程序的其他部分。
- **全局作用域 (Global Scope)**：在**所有函数之外**创建的变量所处的区域。这些变量可以在程序的任何地方被访问（在函数内部也可以读取，但修改需要特殊声明）。全局变量就像“公共资源”，使用时要格外小心。

#### 3. 示例与应用

**例子1：点餐系统**

- **场景**：餐厅的“制作汉堡”函数。
- **代码**：
  ```python
  def make_burger(patty_type, sauce):  # patty_type 和 sauce 是参数
      print(f"正在制作一个{patty_type}汉堡，加入{sauce}酱。")
      return "汉堡做好了！"

  # 调用函数时，"牛肉" 和 "番茄" 是实参
  order1 = make_burger("牛肉", "番茄")
  # 调用函数时，"鸡肉" 和 "沙拉" 是实参
  order2 = make_burger("鸡肉", "沙拉")
  ```
- **解释**：同一个 `make_burger` 函数，通过传入不同的**实参**（牛肉/鸡肉，番茄/沙拉），就能制作出不同的汉堡。`patty_type` 和 `sauce` 这两个**参数**在函数内部工作，不影响外部的变量。

**例子2：计算器**

- **场景**：一个计算矩形面积的函数。
- **代码**：
  ```python
  width = 10  # 全局变量

  def calculate_area(length):
      area = length * width  # 在函数内部读取了全局变量 width
      return area

  print(calculate_area(5))  # 输出 50
  # print(area)  # 这行会报错！因为 area 是局部变量，在函数外无法访问
  ```
- **解释**：`width` 是**全局变量**，可以在函数内部被读取。而 `area` 和 `length` 是**局部变量**，只在 `calculate_area` 函数内部有效。试图在函数外部访问 `area` 会导致错误，这体现了**局部作用域**的隔离性。

#### 4. 常见误区

- **误区一：认为函数内部可以随意修改全局变量**
  - **错误理解**：在函数内部直接写 `x = x + 1`，以为能修改全局变量 `x`。
  - **正确理解**：在函数内部给一个变量赋值时，Python会默认创建一个新的**局部变量**，而不是修改同名的全局变量。如果想修改全局变量，必须使用 `global` 关键字声明。例如：
    ```python
    count = 0
    def increment():
        global count  # 声明要使用全局变量 count
        count = count + 1
    increment()
    print(count)  # 输出 1
    ```

- **误区二：混淆参数和实参的名称**
  - **错误理解**：认为调用函数时传入的变量名必须和函数定义时的参数名一致。
  - **正确理解**：参数名和实参名完全无关。实参只是把它的**值**传递给了参数。例如：
    ```python
    def show_info(name):
        print(f"你好，{name}")

    my_name = "小红"
    show_info(my_name)  # 实参是 my_name，它的值是 "小红"，被传递给了参数 name
    ```
