## 第6单元 函数与模块

### 6.1 函数的定义与调用

#### 1. 核心概念

想象一下，你正在做一顿复杂的晚餐。你需要洗菜、切菜、炒菜、煮饭……如果每做一道菜，你都要重新学习一遍“如何洗菜”、“如何切菜”，那将会非常低效。

在编程中，**函数 (Function)** 就是解决这个问题的“菜谱”。它是一段**组织好的、可重复使用的、用来实现单一或相关联功能的代码块**。

简单来说，函数就像一个“代码积木”。你只需要定义好这个积木（写下它的功能），然后在需要的时候，直接“调用”它（使用它的名字）就可以了。这不仅能避免重复编写相同的代码，还能让你的程序结构更清晰、更容易理解和维护。

#### 2. 关键定义

-   **函数定义 (Function Definition)**：创建函数的过程。你需要告诉Python：
    -   这个函数叫什么名字（函数名）。
    -   它需要什么“原材料”才能工作（**参数 (Parameters)**）。
    -   它要完成什么操作（**函数体 (Function Body)**）。
    -   它最终会产出什么结果（**返回值 (Return Value)**）。
    -   定义函数使用关键字 `def`。

-   **函数调用 (Function Call)**：使用函数的过程。你只需要写出函数的名字，并在括号内提供它需要的“原材料”（**实参 (Arguments)**），程序就会跳转到函数定义的地方去执行代码。

-   **参数 (Parameters)**：定义函数时，写在括号里的变量，用来接收调用时传入的数据。

-   **实参 (Arguments)**：调用函数时，实际传递给函数的值。

-   **返回值 (Return Value)**：函数执行完毕后，通过 `return` 语句返回给调用者的结果。如果没有 `return` 语句，函数默认返回 `None`。

#### 3. 示例与应用

**例子1：制作一杯咖啡**

-   **场景**：你每天都要喝咖啡，每次都需要“磨豆”、“冲泡”、“加奶”等步骤。
-   **函数定义**：定义一个名为 `make_coffee` 的函数，它接受一个参数 `type`（咖啡类型）。
    ```python
    def make_coffee(type):
        print(f"开始制作{type}咖啡...")
        print("1. 磨豆")
        print("2. 冲泡")
        print("3. 加奶")
        print(f"{type}咖啡制作完成！")
        return "一杯热咖啡"
    ```
-   **函数调用**：当你需要咖啡时，只需调用这个函数。
    ```python
    my_coffee = make_coffee("拿铁")  # 调用函数，传入实参"拿铁"
    print(my_coffee)  # 输出：一杯热咖啡
    ```

**例子2：计算圆的面积**

-   **场景**：在数学或物理问题中，经常需要计算圆的面积。
-   **函数定义**：定义一个名为 `circle_area` 的函数，它接受一个参数 `radius`（半径）。
    ```python
    import math
    def circle_area(radius):
        area = math.pi * radius ** 2
        return area
    ```
-   **函数调用**：需要计算时，直接调用。
    ```python
    area_of_circle_5 = circle_area(5)  # 计算半径为5的圆的面积
    print(f"半径为5的圆面积是：{area_of_circle_5:.2f}")  # 输出：半径为5的圆面积是：78.54
    ```

**例子3：检查用户是否成年**

-   **场景**：在注册网站时，需要判断用户年龄是否大于等于18岁。
-   **函数定义**：定义一个名为 `is_adult` 的函数，接受一个参数 `age`（年龄），返回一个布尔值。
    ```python
    def is_adult(age):
        if age >= 18:
            return True
        else:
            return False
    ```
-   **函数调用**：
    ```python
    user_age = 20
    if is_adult(user_age):
        print("欢迎注册！")
    else:
        print("抱歉，您未满18岁。")
    ```

#### 4. 常见误区

1.  **混淆“定义”和“调用”**：很多初学者会认为，写完 `def` 定义了一个函数后，函数里的代码就会自动执行。**这是错误的！** 定义函数只是“告诉”Python有这么个功能，但并不会执行它。只有当你**调用**它（写出函数名和括号）时，里面的代码才会真正运行。

2.  **忘记写括号**：调用函数时，**括号 `()` 是必须的**。即使函数没有参数，也需要写 `()`。例如，`my_function` 只是引用函数本身，而 `my_function()` 才是调用它。忘记括号会导致程序无法按预期执行，甚至报错。

#### 5. 思考题

假设你正在编写一个程序，需要多次计算不同形状（如圆形、矩形、三角形）的面积。你是应该为每种形状都写一个独立的函数，还是写一个包含复杂逻辑的“万能”函数？请从代码的**可读性**、**可维护性**和**复用性**三个角度，分析这两种设计思路的优缺点。
