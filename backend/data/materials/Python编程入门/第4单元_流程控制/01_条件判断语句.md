## 第4单元 流程控制

### 知识点：条件判断语句

#### 1. 核心概念

想象一下，你每天早上醒来，大脑都会进行一系列的判断：“如果今天是周末，我就多睡一会儿；否则，我就得起床去上学或上班。” 这种根据某个条件是否成立，来决定接下来做什么的逻辑，就是**条件判断**。

在编程中，条件判断语句允许程序“思考”和“决策”。它让程序不再是简单地从头执行到尾，而是能够根据不同的情况，执行不同的代码块。这就像给程序装了一个“大脑”，让它能应对各种变化。

#### 2. 关键定义

- **条件判断语句 (Conditional Statement)**：一种编程结构，用于根据一个或多个条件的真假，来决定执行哪一段代码。
- **`if` 语句**：最基本的条件判断语句。如果条件为**真 (True)**，则执行其后的代码块。
- **`else` 语句**：与 `if` 搭配使用。当 `if` 的条件为**假 (False)** 时，执行 `else` 后的代码块。
- **`elif` 语句**：是“else if”的缩写。用于在 `if` 之后，检查另一个条件。可以有多个 `elif`，用于处理多种可能的情况。
- **布尔表达式 (Boolean Expression)**：一个结果为**真 (True)** 或**假 (False)** 的表达式。例如：`5 > 3` (结果为真)，`10 == 20` (结果为假)。
- **代码块 (Code Block)**：在条件判断后，需要缩进（通常是4个空格或一个Tab键）的一组代码。Python通过缩进来识别哪些代码属于同一个条件分支。

#### 3. 示例与应用

**生活例子1：根据天气决定出行方式**

```python
weather = "下雨"  # 假设天气是“下雨”

if weather == "下雨":
    print("带上雨伞！")  # 条件为真，执行此行
else:
    print("天气不错，出门吧！") # 条件为假，执行此行
```

**生活例子2：根据考试成绩评级**

```python
score = 85

if score >= 90:
    print("优秀！")
elif score >= 80:  # 如果第一个条件不满足，检查这个
    print("良好！")
elif score >= 60:
    print("及格！")
else:  # 以上所有条件都不满足
    print("需要加油！")
```

**应用场景：登录验证**

```python
# 用户输入的用户名和密码
input_username = "小明"
input_password = "123456"

# 预设的正确用户名和密码
correct_username = "小明"
correct_password = "123456"

if input_username == correct_username and input_password == correct_password:
    print("登录成功！欢迎回来！")
else:
    print("用户名或密码错误，请重试。")
```

#### 4. 常见误区

- **误区一：混淆 `=` 和 `==`**
  - `=` 是**赋值运算符**，用于将右边的值赋给左边的变量，例如 `a = 5`。
  - `==` 是**比较运算符**，用于判断左右两边的值是否相等，例如 `a == 5` 会返回 `True` 或 `False`。
  - **典型错误**：在 `if` 语句的条件中写成 `if a = 5:`，这会导致语法错误。

- **误区二：忘记缩进**
  - Python 使用缩进来定义代码块。`if`、`elif`、`else` 下面的代码必须统一缩进。
  - **典型错误**：
    ```python
    if 5 > 3:
    print("这是对的")  # 这一行没有缩进，会报错！
    ```
