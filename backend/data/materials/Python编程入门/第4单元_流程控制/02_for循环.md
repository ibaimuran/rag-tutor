## 第4单元 流程控制

### 4.2 for循环

#### 1. 核心概念

想象一下，你需要给班级里的每一位同学发一张试卷。最笨的方法是一个一个地喊名字，喊到谁就给谁。但更聪明、更高效的方法是：“请同学们按顺序，一个一个上来领试卷。” 这个“按顺序，一个一个”的过程，在编程中就是**循环**。

**for循环** 是Python中一种非常强大的循环结构。它的核心思想是：**“对于某个集合中的每一个元素，都执行一遍指定的操作”**。这个“集合”可以是一串数字、一个列表、一个字符串，甚至是文件中的每一行。

简单来说，`for`循环就像一个自动化的“遍历器”，它帮你从头到尾“走”一遍数据，并对沿途遇到的每一个“东西”都做同样的事情，直到走完为止。

#### 2. 关键定义

-   **`for`循环**：一种用于**遍历**（iterate）一个**可迭代对象**（iterable）的循环结构。它会自动获取序列中的下一个元素，直到所有元素都被处理完毕。
-   **可迭代对象**：可以一个接一个地返回其内部元素的对象。常见的可迭代对象包括：**列表**（list）、**字符串**（string）、**元组**（tuple）、**字典**（dictionary）、**范围**（range）等。
-   **循环变量**：在`for`循环中，用于临时存储当前正在被处理的元素的变量。它的值会在每次循环开始时被更新为序列中的下一个元素。
-   **循环体**：被`for`循环重复执行的代码块。通常通过**缩进**（indentation）来表示其属于循环体。

#### 3. 示例与应用

**示例1：逐个点名（遍历列表）**

假设有一个班级名单列表 `students = ["小明", "小红", "小刚", "小丽"]`。老师想对每个同学说“你好”。

```python
students = ["小明", "小红", "小刚", "小丽"]
for student in students:
    print(f"{student}同学，你好！")
```
**输出结果：**
```
小明同学，你好！
小红同学，你好！
小刚同学，你好！
小丽同学，你好！
```
**解释**：`for student in students:` 这行代码的意思是：“对于`students`列表中的每一个元素，把它临时命名为`student`，然后执行下面的缩进代码块”。循环会依次将`student`赋值为“小明”、“小红”...，并执行`print`语句。

**示例2：数数（使用`range()`函数）**

你想从1数到5。`range(1, 6)` 可以生成一个从1开始到5结束的数字序列（注意：`range()`是左闭右开，即包含起始值，不包含结束值）。

```python
for number in range(1, 6):
    print(f"当前数字是：{number}")
```
**输出结果：**
```
当前数字是：1
当前数字是：2
当前数字是：3
当前数字是：4
当前数字是：5
```
**解释**：`range(1, 6)` 创建了一个可迭代对象，它依次包含数字1, 2, 3, 4, 5。循环变量`number`会依次取到这些值。

**示例3：逐字阅读（遍历字符串）**

你可以像遍历列表一样遍历一个字符串中的每一个字符。

```python
word = "Python"
for char in word:
    print(char)
```
**输出结果：**
```
P
y
t
h
o
n
```

#### 4. 常见误区

-   **误区一：忘记缩进循环体**
    Python使用缩进来定义代码块。如果循环体没有正确缩进，程序会报错，或者只有第一行代码被认为是循环的一部分。

    ```python
    # 错误示例
    for i in range(3):
    print(i)  # 这行没有缩进，会报错！
    print("循环结束") # 这行也没有缩进，不属于循环体
    ```

-   **误区二：在循环内部错误地修改正在遍历的列表**
    初学者有时会尝试在遍历列表的同时，向列表中添加或删除元素。这通常会导致意想不到的结果，比如跳过某些元素或产生无限循环。更安全的做法是遍历列表的副本（例如 `for item in list[:]`）或使用列表推导式。
