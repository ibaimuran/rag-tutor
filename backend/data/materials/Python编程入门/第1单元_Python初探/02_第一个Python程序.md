## 第1单元 Python初探

### 知识点：第一个Python程序

欢迎来到Python编程的世界！编写并运行你的第一个程序，是开启这段奇妙旅程的第一步。这就像学习一门新语言时，学会说第一句“你好”一样，简单却意义非凡。

#### 1. 核心概念

想象一下，你正在给一个非常聪明、但只会说特定语言的机器人下达指令。这个机器人就是**计算机**，而它懂的语言就是**编程语言**。Python就是其中一种。

我们的第一个程序，就是要教会这个机器人做一件最简单的事：在屏幕上显示一句话。这个动作在编程里叫做**输出**。我们通过编写一行特定的“咒语”（代码），告诉Python解释器（那个负责理解并执行指令的“翻译官”）：“嘿，请把‘Hello, World!’这句话显示出来。”

这个过程的核心是**编写代码** -> **运行代码** -> **看到结果**。你不需要一开始就理解所有细节，只需要感受这个“下达指令并得到反馈”的完整流程。

#### 2. 关键定义

-   **程序（Program）**：一系列按顺序排列、告诉计算机要做什么的指令集合。我们的第一个程序就是一条指令。
-   **代码（Code）**：用编程语言（如Python）编写的文本，也就是程序的“原材料”。
-   **输出（Output）**：程序运行后，在屏幕上显示出来的信息。这是我们与程序交互的主要方式之一。
-   **`print()` 函数（Function）**：这是Python内置的一个非常常用的工具。它的作用就是将其括号内的内容“打印”或“输出”到屏幕上。**函数**可以理解为一个预先定义好的、能完成特定任务的小机器。
-   **字符串（String）**：在Python中，用**单引号（‘ ’）**或**双引号（“ ”）**括起来的一串文本，比如 `‘Hello, World!’`。它告诉Python：“这是一个文本，不要把它当作代码来执行。”

#### 3. 示例与应用

编写第一个程序不仅仅是仪式感，它背后的“输出”概念无处不在。

1.  **计算器应用**：当你打开手机计算器，输入 `1 + 1` 并按下 `=` 键，屏幕上显示 `2`。这个 `2` 就是计算器程序根据你的输入，通过内部计算后**输出**给你的结果。`print()` 函数就是实现这种“显示结果”功能的最基础工具。

2.  **自动取款机（ATM）**：当你插入银行卡，屏幕上会显示“请输入密码”。这个提示信息就是ATM程序通过**输出**功能显示给你的。程序内部可能使用了类似 `print(“请输入密码”)` 的指令（当然，实际代码会更复杂）。

3.  **游戏中的对话**：在角色扮演游戏中，当主角与NPC（非玩家角色）对话时，屏幕上会弹出对话框，显示NPC说的话。这背后也是程序在不断地**输出**文本信息，让你了解游戏剧情。

#### 4. 常见误区

1.  **忘记引号**：很多初学者会写成 `print(Hello, World!)`。这会导致程序报错，因为Python会把 `Hello` 和 `World` 当作变量名或命令去执行，而不是文本。**记住：要输出的文本必须用引号括起来。**

2.  **混淆中英文符号**：Python代码中所有的标点符号，如括号 `()`、引号 `‘’`、逗号 `,`，都必须是**英文半角**符号。如果使用了中文全角符号（如 `（`、`）`、`‘’`、`，`），程序会无法识别并报错。**这是新手最容易忽略的细节。**

#### 5. 思考题

`print()` 函数除了能输出文本，还能输出数字吗？比如 `print(3 + 5)`，你觉得屏幕上会显示什么？是 `3 + 5` 这个算式本身，还是计算后的结果 `8`？为什么？
