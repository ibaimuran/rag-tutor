## 第2单元 变量与数据类型

### 1. 核心概念

想象一下，你有一个神奇的储物盒。你可以给这个盒子贴上一个标签，比如“我的零食”。然后，你可以把任何东西放进去，比如一包薯片。当你需要薯片时，你只需要找到贴着“我的零食”标签的盒子，打开它，就能拿到薯片。

在Python编程中，**变量**就是这个神奇的“储物盒”，而**赋值**就是“把东西放进去”这个动作。变量是用来存储数据的容器，而赋值则是将数据放入这个容器的过程。变量最重要的特点是：**它的值是可以改变的**。就像你可以把“我的零食”盒子里的薯片换成巧克力一样，变量的值也可以在程序运行过程中被重新赋予新的数据。

### 2. 关键定义

- **变量 (Variable)**：在程序中用来存储数据的**命名**内存空间。你可以把它理解为一个有名字的容器。
- **赋值 (Assignment)**：使用等号 (`=`) 将**值**存储到**变量**中的操作。注意，这里的等号不是数学上的“等于”，而是“把...赋值给...”的意思。
- **变量名 (Variable Name)**：用来标识变量的**名称**，必须遵循一定的命名规则（如不能以数字开头，不能使用Python的**关键字**等）。
- **数据类型 (Data Type)**：变量中存储的数据的种类，例如**整数 (int)**、**浮点数 (float)**、**字符串 (str)** 等。Python会根据你赋的值自动判断数据类型。

### 3. 示例与应用

**例子1：记录学生成绩**

假设老师要记录一个学生的考试成绩。
```python
# 创建一个变量来存储学生的姓名
student_name = "小明"  # 将字符串"小明"赋值给变量student_name

# 创建一个变量来存储数学成绩
math_score = 95  # 将整数95赋值给变量math_score

# 打印出学生的姓名和成绩
print(student_name, "的数学成绩是：", math_score)
```
在这个例子中，`student_name` 和 `math_score` 就是变量，它们分别存储了学生的姓名和成绩。

**例子2：游戏中的生命值**

在游戏中，角色的生命值会不断变化。
```python
# 初始生命值设为100
health = 100
print("当前生命值：", health)

# 角色受到攻击，生命值减少30
health = health - 30  # 将health原来的值减去30，再赋值给health
print("受到攻击后生命值：", health)

# 角色使用恢复药水，生命值增加20
health = health + 20
print("使用药水后生命值：", health)
```
这个例子清晰地展示了变量值可以动态改变的特性。

**例子3：计算购物总价**

去超市购物，需要计算总价。
```python
# 苹果单价5元，买了3斤
apple_price = 5.0  # 浮点数
apple_quantity = 3  # 整数
total_apple_cost = apple_price * apple_quantity

# 香蕉单价4元，买了2斤
banana_price = 4.0
banana_quantity = 2
total_banana_cost = banana_price * banana_quantity

# 计算总价
total_cost = total_apple_cost + total_banana_cost
print("购物总价为：", total_cost, "元")
```
这里，我们用变量存储了单价、数量和总价，并通过赋值和运算得到了最终结果。

### 4. 常见误区

- **误区一：把赋值等号 (`=`) 当作数学等号**
  很多初学者会认为 `x = x + 1` 在数学上是错误的。但在编程中，它的意思是：**先计算等号右边 `x + 1` 的值，然后将这个新值赋值给左边的变量 `x`**。这相当于把变量 `x` 原来的值取出来加1，再放回去。这是程序中最常见的操作之一。

- **误区二：变量名可以随意使用**
  变量名不能以数字开头（如 `1name` 是错误的），也不能包含空格或特殊符号（下划线 `_` 除外）。更重要的是，不能使用Python的**保留关键字**（如 `if`, `for`, `while`, `print` 等）作为变量名，否则程序会报错。
