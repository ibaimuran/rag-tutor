## 第2单元 变量与数据类型

### 2.1 基本数据类型

#### 1. 核心概念

想象一下，你的书桌上放着各种不同的物品：一本写着文字的笔记本、一个计算器上显示的数字、一张写着“是”或“否”的便签。在Python的世界里，数据也像这些物品一样，有着不同的“种类”。**基本数据类型**就是Python用来区分这些不同种类数据的“标签”。它告诉计算机：“这是一段文字，请像处理文字一样对待它”或“这是一个数字，请像处理数字一样计算它”。理解数据类型，是让计算机准确理解我们意图的第一步。

#### 2. 关键定义

Python中最常用的几种基本数据类型如下：

-   **整数 (`int`)**: 表示没有小数部分的数字，例如 `10`、`-5`、`0`、`2024`。它就像我们数数时用的数字。
-   **浮点数 (`float`)**: 表示带有小数部分的数字，例如 `3.14`、`-0.5`、`2.0`。它就像我们测量长度或重量时得到的数值。
-   **字符串 (`str`)**: 表示一段文本，必须用**单引号 (`'`)** 或**双引号 (`"`)** 括起来，例如 `'你好，世界'`、`"Python"`、`'123'`。注意，`'123'` 是字符串，不是数字。
-   **布尔值 (`bool`)**: 表示逻辑上的“真”或“假”，只有两个值：`True` 和 `False`。它就像回答“是”或“否”的问题。

#### 3. 示例与应用

让我们看看这些数据类型在生活中是如何体现的：

1.  **计算购物总价**：
    -   你买了2本书，每本价格是39.90元。
    -   `2` 是一个**整数 (`int`)**，代表书的数量。
    -   `39.90` 是一个**浮点数 (`float`)**，代表书的单价。
    -   计算总价 `2 * 39.90`，结果 `79.8` 也是一个**浮点数 (`float`)**。

2.  **记录个人信息**：
    -   你的名字是“小明”，年龄是12岁，是否已经注册了会员？
    -   `"小明"` 是一个**字符串 (`str`)**。
    -   `12` 是一个**整数 (`int`)**。
    -   `True` 或 `False` 是一个**布尔值 (`bool`)**，用来表示“是会员”或“不是会员”。

3.  **游戏中的状态判断**：
    -   一个游戏角色有100点生命值（整数），受到攻击后生命值减少。
    -   游戏逻辑会判断 `生命值 > 0` 这个条件，这个条件的结果就是一个**布尔值 (`bool`)**：如果生命值大于0，结果为 `True`（角色存活）；否则为 `False`（角色死亡）。

#### 4. 常见误区

-   **误区一：混淆数字和字符串**
    -   **错误理解**：认为 `'123'` 和 `123` 是一样的，可以互相计算。
    -   **正确理解**：`'123'` 是**字符串**，它只是三个字符“1”、“2”、“3”的组合，不能用于数学运算。而 `123` 是**整数**，可以进行加减乘除。尝试 `'123' + 1` 会导致程序报错。

-   **误区二：混淆整数和浮点数**
    -   **错误理解**：认为 `10` 和 `10.0` 在计算机中是完全相同的。
    -   **正确理解**：虽然数值上相等，但 `10` 是**整数 (`int`)**，`10.0` 是**浮点数 (`float`)**。它们在计算机内部的存储方式和运算规则略有不同。例如，整数运算 `10 / 3` 在Python 3中会得到浮点数 `3.3333333333333335`，而不是整数 `3`。
