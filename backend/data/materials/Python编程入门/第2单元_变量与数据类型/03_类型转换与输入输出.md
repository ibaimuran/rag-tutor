## 第2单元 变量与数据类型

### 2.3 类型转换与输入输出

#### 1. 核心概念

想象一下，你有一个装满数字的盒子（`int`类型），但你的朋友只认识文字（`str`类型）。为了让朋友能理解盒子里的内容，你需要把数字“翻译”成文字。在Python中，这种将数据从一种**数据类型**转换成另一种**数据类型**的过程，就叫做**类型转换**。

而**输入输出**，则是程序与外界沟通的桥梁。**输入**是程序从用户那里获取信息（比如你的名字），**输出**则是程序把处理结果展示给你看（比如“你好，张三！”）。Python中最常用的输入函数是`input()`，输出函数是`print()`。

`input()`函数有一个重要的特点：**它获取到的任何数据，都会被当作字符串（`str`）来处理**。这就意味着，如果你想让用户输入一个数字并进行数学运算，就必须先进行类型转换。

#### 2. 关键定义

-   **`input()`函数**：用于从键盘获取用户输入。程序会暂停执行，等待用户输入内容并按下回车键。**返回值**永远是**字符串（`str`）**类型。
-   **`print()`函数**：用于在屏幕上输出信息。它可以输出一个或多个值，值之间默认用空格分隔，输出结束后默认换行。
-   **类型转换**：将一种数据类型转换为另一种数据类型的过程。
-   **`int()`函数**：用于将一个值转换为**整数（`int`）**类型。例如，`int("123")` 会得到整数 `123`。
-   **`float()`函数**：用于将一个值转换为**浮点数（`float`）**类型。例如，`float("3.14")` 会得到浮点数 `3.14`。
-   **`str()`函数**：用于将一个值转换为**字符串（`str`）**类型。例如，`str(100)` 会得到字符串 `"100"`。

#### 3. 示例与应用

**场景一：计算年龄**

你想写一个程序，询问用户的出生年份，然后计算出他今年的年龄。

```python
# 1. 获取用户输入的出生年份（注意：input返回的是字符串）
birth_year_str = input("请输入你的出生年份：")

# 2. 将字符串类型的年份转换为整数类型，才能进行减法运算
birth_year_int = int(birth_year_str)

# 3. 计算年龄（假设今年是2024年）
current_year = 2024
age = current_year - birth_year_int

# 4. 输出结果（注意：age是整数，需要转换为字符串才能与文字拼接）
print("你今年" + str(age) + "岁了！")
```

**场景二：计算平均分**

老师让你写一个程序，输入两门课的成绩，并计算平均分。

```python
# 1. 获取两门课的成绩（都是字符串）
score1_str = input("请输入第一门课的成绩：")
score2_str = input("请输入第二门课的成绩：")

# 2. 将字符串转换为浮点数（因为成绩可能有小数）
score1 = float(score1_str)
score2 = float(score2_str)

# 3. 计算平均分
average = (score1 + score2) / 2

# 4. 输出结果
print("两门课的平均分是：", average)
```

**场景三：数字与字符串的拼接**

你想输出“我有5个苹果”，但“5”这个数字是计算出来的。

```python
apple_count = 5
# 错误做法：print("我有" + apple_count + "个苹果")  # 会报错！
# 正确做法：使用str()将整数转换为字符串
message = "我有" + str(apple_count) + "个苹果"
print(message)
```

#### 4. 常见误区

1.  **忘记转换`input()`的结果**：这是初学者最容易犯的错误。直接对`input()`的结果进行数学运算，会导致程序报错。
    ```python
    # 错误示例
    age = input("请输入你的年龄：")
    next_year_age = age + 1  # 错误！age是字符串，不能和整数相加
    print(next_year_age)
    ```
    **正确做法**：先使用`int()`或`float()`进行转换。

2.  **混淆`int()`和`float()`**：`int("3.14")`会报错，因为`int()`函数不能直接转换带小数点的字符串。应该先使用`float("3.14")`得到`3.14`，如果需要整数，再使用`int(3.14)`得到`3`（注意：这是向下取整）。
