## 第2单元 变量与数据类型

### 知识点：输入与输出

#### 1. 核心概念

想象一下，你和朋友聊天。你说话（**输出**信息），朋友听到后回答你（**输入**信息）。计算机程序也是如此，它需要与外界进行信息交换。

*   **输出**：是程序将处理结果、提示信息等展示给用户的过程。就像你告诉朋友“今天天气真好”。
*   **输入**：是程序从用户那里获取数据的过程。就像你问朋友“你觉得呢？”，然后等待他的回答。

在Python中，最基础的输出方式是`print()`函数，它能把信息显示在屏幕上。最基础的输入方式是`input()`函数，它能让程序暂停，等待用户从键盘输入一些内容，然后程序再继续运行。

#### 2. 关键定义

*   **输出 (Output)**：程序向外部设备（如屏幕、文件、打印机等）发送数据的过程。在Python中，最常用的输出函数是**`print()`**。
*   **输入 (Input)**：程序从外部设备（如键盘、鼠标、文件等）接收数据的过程。在Python中，最常用的输入函数是**`input()`**。
*   **`print()`函数**：用于在控制台（屏幕）上显示文本或变量的值。它可以一次输出多个内容，用逗号分隔，输出时会在内容之间自动添加一个空格。
*   **`input()`函数**：用于从键盘获取用户输入。当程序执行到`input()`时，会暂停并等待用户输入。用户输入完内容并按下回车键后，`input()`会将用户输入的内容作为**字符串（string）** 返回给程序。
*   **字符串 (String)**：由单引号`'`、双引号`"`或三引号`'''`或`"""`括起来的一串文本，是Python中表示文本的数据类型。

#### 3. 示例与应用

**例子1：简单的问候程序**
这是一个最典型的“输入-处理-输出”流程。
```python
# 1. 输出提示信息
print("你好！请问你叫什么名字？")

# 2. 获取用户输入（名字）
name = input()

# 3. 输出个性化的问候
print("很高兴认识你，" + name + "！")
```
**生活场景**：就像你在商店门口，店员说“欢迎光临”（输出），然后问你“请问需要什么？”（等待输入），你回答后，店员会根据你的回答做出下一步行动（输出）。

**例子2：计算年龄**
```python
# 1. 获取用户出生年份（注意：input返回的是字符串）
birth_year_str = input("请输入你的出生年份：")

# 2. 将字符串转换为整数，以便进行数学运算
birth_year = int(birth_year_str)

# 3. 计算年龄
current_year = 2024
age = current_year - birth_year

# 4. 输出结果
print("你今年大约", age, "岁了。")
```
**生活场景**：在医院的挂号机上，你输入自己的出生日期（输入），机器计算出你的年龄（处理），然后显示在屏幕上（输出）。

**例子3：制作一个简单的计算器**
```python
# 获取两个数字
num1 = float(input("请输入第一个数字："))
num2 = float(input("请输入第二个数字："))

# 计算和
sum_result = num1 + num2

# 输出结果
print("两个数的和是：", sum_result)
```
**生活场景**：你在超市购物，收银员扫描商品（输入价格），收银机自动计算总价（处理），然后显示在屏幕上（输出）。

#### 4. 常见误区

*   **误区一：忘记`input()`返回的是字符串**
    很多初学者会直接对`input()`的结果进行数学运算，例如`age = input("请输入年龄：") + 1`，这会导致错误。因为`input()`返回的是字符串，不能直接和数字相加。正确的做法是先用`int()`或`float()`进行类型转换。

    **错误示例**：
    ```python
    age = input("请输入年龄：")  # 假设输入20
    next_year_age = age + 1    # 错误！会报错，因为age是字符串"20"
    ```

    **正确示例**：
    ```python
    age = int(input("请输入年龄："))  # 将输入转换为整数
    next_year_age = age + 1          # 正确，age现在是整数20
    ```

*   **误区二：混淆`print()`输出多个内容时的分隔符**
    `print("a", "b", "c")` 会输出 `a b c`（默认用空格分隔）。如果想输出 `abc`，需要写成 `print("a" + "b" + "c")` 或者使用 `print("a", "b", "c", sep="")`。
