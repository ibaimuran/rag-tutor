## 第10单元 综合项目：简易计算器

### 1. 核心概念

欢迎来到第一个综合项目！在之前的课程中，我们学习了Python的**变量**、**数据类型**、**输入输出**、**条件判断**（`if-elif-else`）等基础知识。现在，我们要把这些零散的知识点像搭积木一样组合起来，制作一个真正能用的程序——**简易计算器**。

这个项目的核心是**程序逻辑**。想象一下，你是一个计算器，用户告诉你两个数字和一个运算符号（比如“加”），你需要判断用户想要做什么，然后执行相应的计算，最后把结果告诉用户。这个过程在程序中就体现为：**接收输入 → 判断指令 → 执行运算 → 输出结果**。通过这个项目，你将学会如何用代码模拟一个真实的、有交互功能的工具。

### 2. 关键定义

- **用户输入（`input()`）**：程序从用户那里获取信息的方式。在计算器中，我们用它来获取用户输入的数字和运算符号。注意，`input()` 获取的数据默认是**字符串（string）** 类型。
- **类型转换（`int()` / `float()`）**：将一种数据类型转换为另一种。因为用户输入的是字符串，不能直接用于数学计算，所以我们需要用 `int()` 或 `float()` 将其转换为**整数（integer）** 或**浮点数（float）**。
- **条件判断（`if-elif-else`）**：程序根据不同的条件执行不同代码块的结构。在计算器中，我们用它来判断用户输入的是“加”、“减”、“乘”还是“除”，并执行对应的运算。
- **算术运算符**：用于执行数学运算的符号，如 `+`（加）、`-`（减）、`*`（乘）、`/`（除）。

### 3. 示例与应用

1.  **生活场景：超市购物**
    你去超市买了2斤苹果和3斤香蕉，想知道一共花了多少钱。你可以打开一个计算器程序，输入“苹果单价 * 2 + 香蕉单价 * 3”，计算器就会帮你算出总价。我们的简易计算器虽然功能简单，但原理相同。

2.  **应用场景：家庭预算**
    你想计算这个月结余了多少钱。你可以输入“收入 - 支出”，计算器会立刻给出结果。这比手动计算要快得多，也避免了出错。

3.  **应用场景：学习辅助**
    你在做数学作业时，可以用计算器快速验证自己的计算结果。比如，计算一个长方形的面积，输入“长 * 宽”，就能得到答案。

### 4. 常见误区

- **误区一：忘记类型转换**
    **错误理解**：认为用户输入的数字可以直接用于计算。
    **正确理解**：`input()` 函数返回的永远是**字符串**。如果你直接写 `num1 + num2`，结果会是两个数字字符串的拼接（例如 `"5" + "3"` 结果是 `"53"`），而不是数学上的加法。**必须**先使用 `int()` 或 `float()` 进行类型转换。

- **误区二：除法的“除零错误”**
    **错误理解**：认为任何除法都能得到结果。
    **正确理解**：在数学中，任何数都不能除以0。在Python中，如果用户输入除数为0，程序会报错并崩溃。一个健壮的程序应该**提前检查**除数是否为0，如果是，则提示用户“除数不能为0”，而不是直接进行除法运算。
