## 第6单元 列表与元组

### 知识点：列表的创建与操作

#### 1. 核心概念

想象一下，你有一个空书包，现在你要去超市购物。你每拿一件商品（比如苹果、牛奶、面包），就把它放进书包里。这个书包就像一个**列表**，它可以按顺序存放多个东西。在Python中，**列表**就是这样一个“容器”，它可以按顺序存放多个数据，比如数字、文字，甚至是其他列表。

列表最大的特点是**有序**且**可变**。有序意味着里面的每个元素都有固定的位置（从0开始编号），可变意味着你可以随时往里面添加新东西、删除旧东西，或者修改里面的某个东西。这就像你的购物清单，你可以随时添加新商品，划掉已买的，或者修改某个商品的数量。

#### 2. 关键定义

-   **列表（List）**：一种Python内置的**数据类型**，用于存储一系列有序的、可变的元素。用方括号 `[]` 表示，元素之间用逗号 `,` 分隔。
-   **元素（Element）**：列表中存储的每一个独立的数据项。例如，在列表 `[‘苹果’， ‘牛奶’]` 中，‘苹果’和‘牛奶’都是元素。
-   **索引（Index）**：列表中每个元素的位置编号。索引从 **0** 开始，而不是1。例如，列表 `[‘苹果’， ‘牛奶’， ‘面包’]` 中，‘苹果’的索引是0，‘牛奶’的索引是1，‘面包’的索引是2。
-   **可变性（Mutable）**：列表的一个重要特性，意味着列表创建后，其内容（元素）可以被修改、增加或删除。这与后面要学的**元组（Tuple）** 正好相反。

#### 3. 示例与应用

**生活例子1：班级点名册**
老师需要一个名单来记录全班同学的名字。这个名单就是一个列表。

```python
# 创建一个空列表，代表一个空的点名册
class_roster = []

# 添加学生名字
class_roster.append(‘张三’)
class_roster.append(‘李四’)
class_roster.append(‘王五’)

print(class_roster)  # 输出：[‘张三’， ‘李四’， ‘王五’]

# 修改第二个同学的名字（索引为1）
class_roster[1] = ‘赵六’
print(class_roster)  # 输出：[‘张三’， ‘赵六’， ‘王五’]

# 删除第一个同学（索引为0）
del class_roster[0]
print(class_roster)  # 输出：[‘赵六’， ‘王五’]
```

**生活例子2：待办事项清单**
你可以用列表来管理每天要做的事情。

```python
# 创建一个待办事项列表
todo_list = [‘写作业’， ‘打扫房间’， ‘给妈妈打电话’]

# 查看第一件事（索引0）
print(“第一件事是：”， todo_list[0])  # 输出：第一件事是： 写作业

# 完成一件事后，从列表中移除
todo_list.remove(‘打扫房间’)
print(“更新后的待办事项：”， todo_list) # 输出：更新后的待办事项： [‘写作业’， ‘给妈妈打电话’]

# 添加新的事项
todo_list.append(‘看一集动画片’)
print(“最终待办事项：”， todo_list) # 输出：最终待办事项： [‘写作业’， ‘给妈妈打电话’， ‘看一集动画片’]
```

**应用场景：存储游戏中的角色状态**
在游戏开发中，可以用列表来存储玩家的背包物品、技能列表或角色坐标。

```python
# 存储玩家背包中的物品
inventory = [‘木剑’， ‘药水’， ‘金币’]

# 存储角色在地图上的坐标 (x, y)
player_position = [100, 200]

# 修改坐标
player_position[0] = 150
print(“玩家当前位置：”， player_position) # 输出：玩家当前位置： [150, 200]
```

#### 4. 常见误区

-   **误区一：索引从1开始。**
    -   **错误理解**：认为列表的第一个元素位置是1。
    -   **正确理解**：Python中所有序列（包括列表、字符串、元组）的索引都是从 **0** 开始的。所以第一个元素的索引是0，第二个是1，以此类推。记住“从0开始数”是避免索引错误的关键。

-   **误区二：混淆 `append()` 和 `extend()`。**
    -   **错误理解**：认为 `append()` 可以一次性添加多个元素。
    -   **正确理解**：`append()` 方法是将一个元素（可以是数字、字符串，甚至是另一个列表）作为一个整体添加到列表末尾。而 `extend()` 方法是将一个可迭代对象（如另一个列表）中的每个元素逐一添加到列表末尾。
    -   **例子**：
        ```python
        list_a = [1, 2]
        list_b = [3, 4]

        list_a.append(list_b)  # 将 list_b 作为一个元素添加
        print(list_a)  # 输出：[1, 2, [3, 4]]  （注意，这是一个嵌套列表）

        list_c = [1, 2]
        list_c.extend(list_b)  # 将 list_b 中的每个元素逐一添加
        print(list_c)  # 输出：[1, 2, 3, 4]  （元素被展开了）
        ```
