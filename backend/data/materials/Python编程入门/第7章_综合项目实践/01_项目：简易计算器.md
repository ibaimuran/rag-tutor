## 第7章 综合项目实践

### 项目：简易计算器

#### 1. 核心概念

想象一下，你手里有一个普通的计算器，按下“1”、“+”、“2”、“=”，它就会显示“3”。这个简单的过程，背后其实隐藏着一个计算机程序的核心工作模式：**输入**、**处理**、**输出**。

*   **输入**：你按下数字和运算符按钮，就是给计算器下达指令。
*   **处理**：计算器内部的芯片（也就是程序）接收到指令后，会进行加法运算。
*   **输出**：计算器将运算结果显示在屏幕上。

我们即将编写的“简易计算器”程序，就是要在Python中模拟这个过程。它将接收用户输入的两个数字和一个运算符（如 `+`, `-`, `*`, `/`），然后根据运算符执行相应的数学运算，最后将结果打印出来。这个项目将把你之前学过的**变量**、**数据类型**、**输入输出**、**条件判断**（`if-elif-else`）等知识点串联起来，是一次非常棒的实战演练。

#### 2. 关键定义

*   **变量 (Variable)**：用来存储数据的“容器”。在我们的计算器中，`num1`、`num2`、`operator` 都是变量，分别用来存放第一个数字、第二个数字和运算符。
*   **数据类型 (Data Type)**：变量中存储的数据的种类。例如，`1`、`2` 是**整数 (int)**，`3.14` 是**浮点数 (float)**，而 `+`、`-` 是**字符串 (string)**。在计算器中，我们需要将用户输入的字符串数字（如 `"10"`）转换为数字类型（如 `10`）才能进行数学运算。
*   **条件判断 (Conditional Statement)**：程序根据某个条件是否成立，来决定执行哪一段代码。在计算器中，我们用 `if` 语句来判断用户输入的运算符是 `+`、`-`、`*` 还是 `/`，然后执行对应的运算。
*   **函数 (Function)**：组织好的、可重复使用的、用来实现特定功能的代码块。Python内置的 `input()` 函数用于获取用户输入，`print()` 函数用于输出结果。我们也可以自己定义函数，比如 `calculate()`，来封装整个计算逻辑。

#### 3. 示例与应用

1.  **生活中的小账本**：你和朋友一起吃饭，花了85元，朋友付了50元，你想快速算出自己应该付多少。你可以编写一个简易计算器，输入 `85`、`-`、`50`，程序立刻输出 `35`。

2.  **网购折扣计算**：你看中一件原价199元的衣服，打7折。你可以用计算器输入 `199`、`*`、`0.7`，快速得到折后价 `139.3` 元。

3.  **平均分计算**：你这次考试语文92分，数学88分，英语95分。你可以用计算器分两步：先输入 `92`、`+`、`88`、`+`、`95` 得到总分，再输入总分、`/`、`3` 得到平均分。当然，更高级的程序可以一次完成。

#### 4. 常见误区

1.  **忘记转换数据类型**：`input()` 函数获取的用户输入永远是**字符串**。如果你直接写 `num1 + num2`，结果会是 `"12" + "34" = "1234"`（字符串拼接），而不是数字相加的 `46`。**必须使用 `int()` 或 `float()` 函数将输入的字符串转换为数字类型。**

    **错误示例**：
    ```python
    num1 = input("请输入第一个数字：")  # 用户输入 12
    num2 = input("请输入第二个数字：")  # 用户输入 34
    result = num1 + num2  # 结果是 "1234"，而不是 46
    print(result)
    ```

    **正确示例**：
    ```python
    num1 = float(input("请输入第一个数字："))  # 转换为浮点数
    num2 = float(input("请输入第二个数字："))  # 转换为浮点数
    result = num1 + num2  # 结果是 46.0
    print(result)
    ```

2.  **除数为零的错误**：在数学中，任何数都不能除以0。如果用户输入了 `/` 作为运算符，并且第二个数字是 `0`，程序会报错并崩溃。**在编写除法运算时，一定要先判断除数是否为0，如果是，则提示用户“除数不能为0”，而不是直接进行运算。**
