## 第3章 数据结构

### 集合与字符串进阶

#### 1. 核心概念

在前面的学习中，我们已经接触了列表和元组这些有序的数据容器。现在，我们将学习两种功能更强大的数据类型：**集合 (Set)** 和**字符串 (String)** 的进阶用法。

*   **集合**就像一个“无序的、不重复的元素袋子”。你可以往里面扔东西，但袋子会自动帮你把重复的东西扔掉，并且里面的东西没有固定的顺序。它的核心价值在于**去重**和**快速判断成员是否存在**。
*   **字符串**的进阶用法，则像是学习如何对一段文字进行“精细加工”。我们不再只是简单地存储一个句子，而是要学会如何**查找**、**替换**、**分割**和**判断**字符串中的内容，就像用一把瑞士军刀对文本进行各种操作。

简单来说，集合解决的是“有没有”和“去重”的问题，而字符串进阶解决的是“如何高效处理文本”的问题。

#### 2. 关键定义

*   **集合 (Set)**：一个**无序**的、**不包含重复元素**的数据结构。使用花括号 `{}` 或 `set()` 函数创建。例如：`{1, 2, 3}` 或 `set("hello")`。
*   **元素唯一性 (Uniqueness)**：集合的核心特性。向集合中添加已存在的元素不会产生任何效果。
*   **集合运算 (Set Operations)**：集合支持数学上的集合运算，如**并集 (Union, `|`)**、**交集 (Intersection, `&`)**、**差集 (Difference, `-`)** 和**对称差集 (Symmetric Difference, `^`)**。
*   **字符串方法 (String Methods)**：Python 为字符串提供了大量内置方法，用于处理文本。例如：
    *   `.find()` / `.index()`：查找子串位置。
    *   `.replace()`：替换子串。
    *   `.split()`：按指定分隔符将字符串分割成列表。
    *   `.join()`：将列表中的元素用指定字符串连接起来。
    *   `.strip()`：去除字符串首尾的空白字符。
    *   `.startswith()` / `.endswith()`：判断字符串是否以某子串开头或结尾。

#### 3. 示例与应用

1.  **学生选课系统 (集合应用)**
    学校要统计所有选修了“Python编程”和“数据结构”两门课的学生名单。
    *   `python_students = {"张三", "李四", "王五", "赵六"}`
    *   `data_structure_students = {"李四", "赵六", "钱七", "孙八"}`
    *   同时选修两门课的学生（**交集**）：`python_students & data_structure_students` 结果为 `{"李四", "赵六"}`。
    *   只选修了一门课的学生（**对称差集**）：`python_students ^ data_structure_students` 结果为 `{"张三", "王五", "钱七", "孙八"}`。

2.  **用户输入清洗 (字符串进阶应用)**
    用户注册时输入了一个带有多余空格的邮箱地址：`"  user@example.com  "`。
    *   使用 `.strip()` 方法可以轻松去除首尾空格：`"  user@example.com  ".strip()` 得到 `"user@example.com"`。
    *   然后使用 `.find("@")` 检查是否包含 `@` 符号，验证邮箱格式。

3.  **日志文件分析 (字符串进阶应用)**
    假设我们有一个服务器日志字符串：`"2023-10-27 10:30:00 ERROR: Database connection failed"`。
    *   使用 `.split(": ")` 可以将它按 `": "` 分割成 `['2023-10-27 10:30:00 ERROR', 'Database connection failed']`。
    *   再使用 `.startswith("ERROR")` 可以快速判断该日志是否为错误日志。

#### 4. 常见误区

1.  **混淆集合和字典**
    *   **错误理解**：认为 `{}` 创建的是一个空集合。
    *   **正确理解**：在 Python 中，`{}` 创建的是一个**空字典**。要创建空集合，必须使用 `set()`。例如：`empty_set = set()`。

2.  **忘记字符串是不可变的 (Immutable)**
    *   **错误理解**：认为 `my_string = "hello"` 后，可以像修改列表一样修改它，例如 `my_string[0] = "H"`。
    *   **正确理解**：字符串是**不可变**的。所有看似“修改”字符串的方法（如 `.replace()`），实际上都是**创建并返回了一个新的字符串**，原字符串保持不变。例如：
        ```python
        my_string = "hello"
        new_string = my_string.replace("h", "H")
        print(my_string)  # 输出: hello (原字符串不变)
        print(new_string) # 输出: Hello (新字符串)
        ```
