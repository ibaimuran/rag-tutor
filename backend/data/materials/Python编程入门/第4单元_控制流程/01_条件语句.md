## 第4单元 控制流程

### 4.1 条件语句

#### 1. 核心概念

想象一下，你每天的生活都充满了各种“如果...就...”的决策。比如，“如果今天下雨，我就带伞；否则，我就不带。” 在编程中，我们也需要让程序根据不同的情况，执行不同的操作。**条件语句**就是用来实现这种“判断”和“分支”功能的代码结构。

它允许程序“思考”：检查某个条件是**真**还是**假**，然后根据结果，选择执行哪一段代码。这就像程序拥有了一个简单的“大脑”，可以根据不同的输入或状态做出不同的反应。

#### 2. 关键定义

-   **条件语句 (Conditional Statement)**：一种编程结构，允许程序根据一个或多个条件的真假，选择性地执行不同的代码块。
-   **布尔表达式 (Boolean Expression)**：一个能够被求值为 `True`（真）或 `False`（假）的表达式。例如：`5 > 3` 的结果是 `True`，`10 == 20` 的结果是 `False`。
-   **`if` 语句**：最基本的条件语句。如果其后的**布尔表达式**为 `True`，则执行其下方的缩进代码块。
-   **`else` 语句**：与 `if` 语句配合使用。当 `if` 的条件为 `False` 时，执行 `else` 下方的缩进代码块。
-   **`elif` 语句**：“else if”的缩写。用于在 `if` 之后检查多个条件。只有当所有前面的条件都为 `False` 时，才会检查 `elif` 的条件。
-   **代码块 (Code Block)**：在条件语句中，通过**缩进**（通常是4个空格或一个Tab键）来标识的一组相关代码。Python 使用缩进来定义代码的归属。

#### 3. 示例与应用

**示例1：判断成绩是否及格**

这是一个非常贴近学生生活的例子。

```python
# 获取用户输入的成绩
score = int(input("请输入你的考试成绩："))

# 条件判断
if score >= 60:
    print("恭喜你，考试及格了！")
else:
    print("很遗憾，考试不及格，需要加油哦！")
```

**应用场景**：在线考试系统自动判断考生是否通过，并给出相应提示。

**示例2：根据天气决定活动**

这个例子展示了 `if-elif-else` 结构，用于处理多种情况。

```python
weather = input("今天天气怎么样？(晴天/下雨/下雪)：")

if weather == "晴天":
    print("我们去公园放风筝吧！")
elif weather == "下雨":
    print("待在家里看电影吧。")
elif weather == "下雪":
    print("出去堆雪人！")
else:
    print("天气未知，还是先看看天气预报吧。")
```

**应用场景**：智能家居系统根据传感器检测到的天气状况，自动调整室内灯光、窗帘或播放音乐。

**示例3：登录验证**

这是网络应用中非常常见的场景。

```python
# 预设的用户名和密码
correct_username = "admin"
correct_password = "123456"

# 获取用户输入
input_username = input("请输入用户名：")
input_password = input("请输入密码：")

# 验证
if input_username == correct_username and input_password == correct_password:
    print("登录成功！欢迎回来。")
else:
    print("用户名或密码错误，请重试。")
```

**应用场景**：任何需要用户登录的网站或App，如邮箱、社交媒体、在线银行等。

#### 4. 常见误区

-   **误区一：混淆 `=` 和 `==`**
    -   **错误理解**：在条件判断中，使用 `=` 来比较两个值是否相等。
    -   **正确做法**：`=` 是**赋值运算符**，用于将右边的值赋给左边的变量。`==` 是**比较运算符**，用于判断左右两边的值是否相等。
    -   **错误示例**：`if score = 100:` （这会导致语法错误，因为 `=` 不能用在条件表达式中）
    -   **正确示例**：`if score == 100:`

-   **误区二：忘记冒号 `:`**
    -   **错误理解**：在 `if`、`elif`、`else` 语句的末尾忘记写冒号。
    -   **正确做法**：`if`、`elif`、`else` 语句的末尾必须有一个英文冒号 `:`，它告诉 Python 接下来是一个代码块。
    -   **错误示例**：`if score >= 60` （缺少冒号，会引发语法错误）
    -   **正确示例**：`if score >= 60:`
