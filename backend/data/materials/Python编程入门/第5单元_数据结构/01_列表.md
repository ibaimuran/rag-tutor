## 第5单元 数据结构

### 列表 (List)

#### 1. 核心概念

想象一下，你有一个购物清单，上面按顺序写着“牛奶、面包、鸡蛋、苹果”。这个清单就是一个典型的**列表**。在Python编程中，**列表**是一种最常用、最灵活的数据结构，它允许你将多个相关的数据项有序地组织在一起，就像一个可以存放各种物品的“容器”。

列表的核心思想是**有序**和**可变**。有序意味着列表中的每个元素都有一个固定的位置（索引），你可以通过这个位置来访问它。可变则意味着你可以随时向列表中添加、删除或修改其中的元素，非常灵活。

#### 2. 关键定义

- **列表 (List)**：一种有序的、可变的、用方括号 `[]` 括起来的元素集合。元素之间用逗号 `,` 分隔。
- **元素 (Element)**：列表中的每一个数据项，可以是任何数据类型（如整数、字符串、甚至另一个列表）。
- **索引 (Index)**：列表中每个元素的位置编号。**索引从0开始**，即第一个元素的索引是0，第二个是1，依此类推。你也可以使用**负索引**，`-1` 表示最后一个元素，`-2` 表示倒数第二个。
- **可变性 (Mutability)**：列表创建后，其内容可以被修改（增、删、改）。这是列表与**元组 (Tuple)** 的主要区别之一。

#### 3. 示例与应用

**生活例子1：学生成绩管理**
假设你要记录班级里5位同学的数学成绩。你可以创建一个列表来存储这些分数。
```python
# 创建一个包含5个整数（成绩）的列表
scores = [95, 87, 92, 78, 88]

# 访问第三个同学的成绩（索引为2）
print(scores[2])  # 输出：92

# 修改最后一个同学的成绩（索引为-1）
scores[-1] = 90
print(scores)  # 输出：[95, 87, 92, 78, 90]
```

**生活例子2：待办事项清单**
你可以用列表来管理一天的任务。
```python
# 创建一个字符串列表
todo_list = ["买早餐", "写报告", "开会", "健身"]

# 在列表末尾添加一个新任务
todo_list.append("取快递")
print(todo_list)  # 输出：['买早餐', '写报告', '开会', '健身', '取快递']

# 删除第二个任务（索引为1）
del todo_list[1]
print(todo_list)  # 输出：['买早餐', '开会', '健身', '取快递']
```

**应用场景：数据分析中的特征向量**
在数据分析中，一个样本的多个特征（如身高、体重、年龄）通常会被存储为一个列表，方便进行批量计算和处理。

#### 4. 常见误区

- **误区一：混淆索引起始值**
  很多初学者会误以为列表的第一个元素索引是1。**请务必记住，Python列表的索引是从0开始的**。访问 `list[1]` 获取的是第二个元素，而不是第一个。

- **误区二：混淆列表与数组 (Array)**
  虽然列表和数组都用于存储序列数据，但它们是不同的。Python的列表可以包含**不同类型**的元素（例如 `[1, "hello", 3.14]`），而大多数编程语言中的数组要求所有元素类型相同。列表更灵活，但某些高性能计算场景下，数组（如NumPy数组）效率更高。
