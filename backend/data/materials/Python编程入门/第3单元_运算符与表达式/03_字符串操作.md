## 第3单元 运算符与表达式

### 3.4 字符串操作

#### 1. 核心概念

想象一下，你正在编辑一条发给朋友的短信。你可以把几个词语拼在一起组成一句话，可以复制粘贴某句话，也可以看看这条短信一共有多少个字。在Python编程中，**字符串（String）** 就是用来表示和处理这类文本数据的。而**字符串操作**，就是教我们如何像编辑短信一样，对文本进行拼接、复制、查找长度等处理。它是编程中最基础、最常用的技能之一。

#### 2. 关键定义

- **字符串（String）**：用单引号 `' '`、双引号 `" "` 或三引号 `''' '''` 括起来的任意文本。例如：`'Hello'`、`"Python编程"`。
- **字符串拼接（Concatenation）**：使用 **`+`** 运算符将两个或多个字符串连接成一个新的字符串。
- **字符串重复（Repetition）**：使用 **`*`** 运算符将一个字符串重复指定的次数。
- **字符串长度（Length）**：使用内置函数 **`len()`** 获取字符串中字符的个数（包括空格和标点）。
- **索引（Index）**：字符串中的每个字符都有一个位置编号，从 **0** 开始。例如，字符串 `"Python"` 中，`'P'` 的索引是0，`'y'` 的索引是1，以此类推。

#### 3. 示例与应用

1.  **制作欢迎标语（字符串拼接）**
    - 场景：一个网站需要根据用户的名字显示个性化的欢迎信息。
    - 代码：
      ```python
      user_name = "小明"
      welcome_message = "欢迎你，" + user_name + "！"
      print(welcome_message)  # 输出：欢迎你，小明！
      ```

2.  **生成分隔线（字符串重复）**
    - 场景：在控制台输出一个美观的分隔线，将不同部分的内容隔开。
    - 代码：
      ```python
      separator = "-" * 30
      print("用户信息")
      print(separator)  # 输出：------------------------------
      print("姓名：张三")
      ```

3.  **检查密码长度（字符串长度）**
    - 场景：注册账号时，系统需要检查用户输入的密码是否满足最低长度要求（例如8位）。
    - 代码：
      ```python
      password = input("请输入密码：")
      if len(password) < 8:
          print("密码长度不足，请至少输入8个字符。")
      else:
          print("密码设置成功！")
      ```

#### 4. 常见误区

- **误区一：混淆数字和字符串**
  - **错误理解**：认为 `"5" + "3"` 的结果是 `8`。
  - **正确理解**：`"5"` 和 `"3"` 都是字符串，`+` 运算符执行的是**拼接**操作，结果是 `"53"`。要得到数学上的8，需要使用数字类型 `5 + 3`。

- **误区二：忘记索引从0开始**
  - **错误理解**：认为字符串 `"Python"` 的第一个字符 `'P'` 的索引是1。
  - **正确理解**：在Python（以及大多数编程语言）中，索引是从 **0** 开始的。所以 `"Python"[0]` 得到 `'P'`，`"Python"[1]` 得到 `'y'`。
