## 第3单元 运算符与表达式

### 3.2 比较与逻辑运算符

在编程的世界里，程序经常需要做出“判断”。比如，判断一个数字是否大于另一个，或者判断两个条件是否同时成立。**比较运算符**和**逻辑运算符**就是用来完成这些判断任务的“决策工具”。它们的结果只有两种：`True`（真）或 `False`（假），这在编程中被称为**布尔值**。

#### 1. 核心概念

想象一下，你是一个裁判。**比较运算符**就像一把尺子，用来衡量两个事物之间的关系（比如谁大、谁小、是否相等）。而**逻辑运算符**则像一个“逻辑门”，它能把多个简单的判断（比如“今天下雨”和“我带了伞”）组合起来，得出一个更复杂的结论（比如“我不会被淋湿”）。

简单来说：
*   **比较运算符**：回答“是”或“否”的问题。
*   **逻辑运算符**：回答“如果...并且...”、“如果...或者...”等组合问题。

#### 2. 关键定义

*   **比较运算符 (Comparison Operators)**：用于比较两个值之间的关系，并返回一个**布尔值** (`True` 或 `False`)。
    *   `==`：**等于**。检查两个值是否相等。**注意**：是两个等号，不要与赋值符号 `=` 混淆。
    *   `!=`：**不等于**。检查两个值是否不相等。
    *   `>`：**大于**。
    *   `<`：**小于**。
    *   `>=`：**大于等于**。
    *   `<=`：**小于等于**。
*   **逻辑运算符 (Logical Operators)**：用于组合多个布尔值或条件，并返回一个最终的**布尔值**。
    *   `and`：**与**。当所有条件都为 `True` 时，结果才为 `True`。可以理解为“并且”。
    *   `or`：**或**。只要有一个条件为 `True`，结果就为 `True`。可以理解为“或者”。
    *   `not`：**非**。对一个布尔值取反。如果条件是 `True`，`not` 后变成 `False`；反之亦然。可以理解为“不”。

#### 3. 示例与应用

1.  **游戏中的生命值判断**
    ```python
    # 假设玩家的生命值 (health) 是 50
    health = 50
    max_health = 100

    # 判断玩家是否满血
    is_full_health = (health == max_health)  # 结果为 False
    print(is_full_health)  # 输出: False

    # 判断玩家是否还活着 (生命值大于0)
    is_alive = (health > 0)  # 结果为 True
    print(is_alive)  # 输出: True
    ```

2.  **网购优惠活动**
    ```python
    # 假设用户购物金额 (amount) 是 150 元，并且是会员 (is_member)
    amount = 150
    is_member = True

    # 判断是否满足“满100减20”的优惠条件 (金额大于等于100 并且 是会员)
    can_get_discount = (amount >= 100) and is_member
    print(can_get_discount)  # 输出: True (两个条件都满足)

    # 判断是否满足“全场包邮”的条件 (金额大于200 或者 是会员)
    can_free_shipping = (amount > 200) or is_member
    print(can_free_shipping)  # 输出: True (因为是会员，所以满足)
    ```

3.  **登录验证**
    ```python
    # 假设用户输入的用户名和密码
    input_username = "admin"
    input_password = "123456"

    # 正确的用户名和密码
    correct_username = "admin"
    correct_password = "abc123"

    # 判断登录是否成功 (用户名正确 并且 密码正确)
    login_success = (input_username == correct_username) and (input_password == correct_password)
    print(login_success)  # 输出: False (因为密码不匹配)

    # 判断是否输入了用户名 (用户名不等于空字符串)
    has_username = input_username != ""
    print(has_username)  # 输出: True
    ```

#### 4. 常见误区

1.  **混淆 `=` 和 `==`**
    *   **错误理解**：认为 `if x = 5:` 可以判断 `x` 是否等于5。
    *   **正确理解**：`=` 是**赋值运算符**，用于将右边的值赋给左边的变量。`==` 才是**比较运算符**，用于判断左右两边是否相等。在条件判断中，必须使用 `==`。

2.  **对 `and` 和 `or` 的误用**
    *   **错误理解**：认为 `if age > 10 and < 20:` 可以判断年龄在10到20之间。
    *   **正确理解**：逻辑运算符 `and` 必须连接两个完整的条件表达式。正确的写法是 `if age > 10 and age < 20:`。每个条件都必须是一个完整的比较表达式。
