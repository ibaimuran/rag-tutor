## 第3单元 运算符与表达式

### 3.1 算术运算符

#### 1. 核心概念

想象一下，你是一个正在做数学题的小学生。你需要进行**加、减、乘、除**这些基本运算。在Python的世界里，计算机就像一个超级计算器，它也需要通过特定的符号来执行这些运算。这些用于执行数学运算的符号，就叫做**算术运算符**。

简单来说，**算术运算符**就是用来对数字（整数或小数）进行数学计算的符号。它们构成了**算术表达式**，比如 `3 + 5` 就是一个表达式，其中 `+` 是运算符，`3` 和 `5` 是操作数。计算机计算这个表达式后，会得到一个结果 `8`。

#### 2. 关键定义

- **算术运算符 (Arithmetic Operators)**：用于执行基本数学运算的符号。Python中常用的有：
    - `+` (加号)：用于**加法**运算。
    - `-` (减号)：用于**减法**运算。
    - `*` (星号)：用于**乘法**运算。
    - `/` (斜杠)：用于**除法**运算，结果总是返回一个**浮点数**（小数）。
    - `//` (双斜杠)：用于**整除**运算，结果只保留商的**整数部分**（向下取整）。
    - `%` (百分号)：用于**取模**运算，即求**余数**。
    - `**` (双星号)：用于**幂运算**，即计算一个数的几次方。

- **操作数 (Operands)**：运算符两侧参与运算的数值。例如在 `10 + 5` 中，`10` 和 `5` 就是操作数。

- **表达式 (Expression)**：由运算符和操作数组成的、能计算出结果的代码片段。例如 `(2 + 3) * 4`。

- **浮点数 (Float)**：带有小数点的数字，如 `3.14`、`2.0`。在Python中，除法运算的结果通常是浮点数。

#### 3. 示例与应用

让我们看看算术运算符在生活中的应用：

1.  **计算购物总价**
    你去超市买了3瓶单价为2.5元的牛奶，和1个单价为15元的面包。计算总价：
    ```python
    milk_price = 2.5
    milk_count = 3
    bread_price = 15
    total = milk_price * milk_count + bread_price
    print(total)  # 输出：22.5
    ```
    这里，`*` 用于计算牛奶总价，`+` 用于将牛奶和面包的价格相加。

2.  **判断奇偶性**
    老师让你判断一个数字是奇数还是偶数。你可以用**取模运算符 `%`** 来求余数。如果一个数除以2的余数是0，它就是偶数；否则是奇数。
    ```python
    number = 17
    remainder = number % 2
    print(remainder)  # 输出：1
    # 因为余数是1，所以17是奇数
    ```

3.  **分配任务**
    班级有45个同学，要分成每组4人进行小组活动。你能算出可以分成几个完整的组，以及剩下几个人吗？
    ```python
    total_students = 45
    group_size = 4
    full_groups = total_students // group_size  # 整除，得到11组
    remaining = total_students % group_size     # 取模，得到1人
    print(f"可以分成 {full_groups} 组，还剩 {remaining} 人。")
    # 输出：可以分成 11 组，还剩 1 人。
    ```

#### 4. 常见误区

- **误区一：混淆 `/` 和 `//`**
    - **错误理解**：认为 `7 / 2` 的结果是 `3`。
    - **正确理解**：`7 / 2` 的结果是 `3.5`（浮点数）。只有 `7 // 2` 的结果才是 `3`（整数）。记住：**一个斜杠是精确除法，两个斜杠是整数除法**。

- **误区二：忘记运算符优先级**
    - **错误理解**：认为 `2 + 3 * 4` 的结果是 `20`。
    - **正确理解**：Python遵循数学中的运算优先级规则：**先乘除，后加减**。所以 `2 + 3 * 4` 会先计算 `3 * 4 = 12`，再计算 `2 + 12 = 14`。如果想先做加法，必须使用括号：`(2 + 3) * 4 = 20`。
