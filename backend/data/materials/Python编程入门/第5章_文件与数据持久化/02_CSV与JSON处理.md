## 第5章 文件与数据持久化

### 5.3 CSV与JSON处理

#### 1. 核心概念

想象一下，你有一张记录全班同学成绩的**表格**，或者一份记录你与朋友聊天记录的**通讯录**。在计算机世界里，我们需要一种标准、通用的方式来存储和交换这类结构化数据。

**CSV** 和 **JSON** 就是两种最流行的“数据格式”，它们就像两种不同的“语言”，用来描述和组织数据，让不同的程序（比如Python、Excel、网页）都能看懂并互相交流。

- **CSV** 擅长处理像电子表格一样的**表格型数据**，它用逗号来分隔每一列的数据。
- **JSON** 则更擅长描述复杂的、有层级关系的数据，比如一个“人”对象，里面包含姓名、年龄、地址（地址本身又是一个包含城市、街道的对象）。

Python提供了内置的 `csv` 和 `json` 模块，让我们可以轻松地读取（**解析**）和写入（**序列化**）这两种格式的数据。

#### 2. 关键定义

- **CSV (Comma-Separated Values)**：一种以纯文本形式存储表格数据的文件格式。每一行代表一条记录，行内的字段（列）通常用逗号分隔。
- **JSON (JavaScript Object Notation)**：一种轻量级的数据交换格式。它基于JavaScript语法，但独立于语言，易于人阅读和编写，也易于机器解析和生成。其结构类似于Python中的字典（`dict`）和列表（`list`）。
- **解析 (Parsing)**：将一种格式的数据（如CSV或JSON文本）转换成程序可以操作的数据结构（如Python的列表、字典）的过程。
- **序列化 (Serialization)**：将程序中的数据结构（如Python的列表、字典）转换成可以存储或传输的格式（如CSV或JSON文本）的过程。

#### 3. 示例与应用

1.  **学生成绩管理 (CSV应用)**
    老师将全班同学的姓名、语文、数学、英语成绩保存在一个 `scores.csv` 文件中。Python程序可以读取这个文件，计算每个学生的总分和平均分，或者分析全班各科的平均成绩。

    ```csv
    # scores.csv 文件内容
    姓名,语文,数学,英语
    张三,90,85,92
    李四,88,95,78
    王五,76,82,91
    ```

2.  **网站API数据交换 (JSON应用)**
    当你用手机App查询天气时，App会向天气服务器发送请求。服务器返回的天气信息（如温度、湿度、未来几天的预报）通常就是JSON格式。Python程序可以解析这个JSON，提取出“今天最高温度”并显示在屏幕上。

    ```json
    // 天气API返回的JSON数据
    {
      "city": "北京",
      "temperature": 25,
      "humidity": 60,
      "forecast": [
        {"day": "周一", "high": 28, "low": 18},
        {"day": "周二", "high": 26, "low": 17}
      ]
    }
    ```

3.  **配置文件 (JSON应用)**
    很多软件使用JSON文件作为配置文件。例如，一个游戏程序可以将用户自定义的“音量大小”、“画面质量”、“快捷键设置”等保存为一个 `config.json` 文件。下次启动游戏时，程序读取这个JSON文件，就能恢复用户的个性化设置。

#### 4. 常见误区

- **误区一：认为CSV只能用逗号分隔。**
    **正解：** 虽然叫“逗号分隔值”，但有时数据本身包含逗号（例如地址“北京市,朝阳区”），这时就需要用引号将整个字段括起来。Python的 `csv` 模块会自动处理这些复杂情况，但初学者手动处理时容易出错。

- **误区二：混淆JSON对象和Python字典。**
    **正解：** JSON的语法看起来很像Python的字典，但它们是不同的东西。JSON是一种**文本格式**，而Python字典是**内存中的数据结构**。例如，JSON的布尔值必须是小写的 `true` 和 `false`，而Python中是 `True` 和 `False`；JSON的字符串必须用双引号 `"`，而Python可以用单引号 `'`。`json` 模块的作用就是在这两者之间进行转换。

#### 5. 思考题

假设你有一个Python列表，里面包含100个字典，每个字典代表一个商品（包含名称、价格、库存）。现在需要将这个列表保存到文件中，以便下次程序运行时能恢复这些数据。你会选择CSV格式还是JSON格式？请从**数据结构的复杂性**和**数据读取的便捷性**两个角度分析你的选择。
