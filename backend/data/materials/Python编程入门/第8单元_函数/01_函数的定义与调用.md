## 第8单元 函数

### 1. 核心概念

想象一下，你正在做一道复杂的菜，比如“红烧肉”。你需要准备食材、切肉、焯水、炒糖色、慢炖……每一步都很繁琐。如果你每次想吃红烧肉都要重复所有这些步骤，那会非常麻烦。

在编程中，**函数** 就像是一个“菜谱”。它把一系列完成特定任务的指令（比如“切肉”、“焯水”、“慢炖”）打包在一起，并给它取一个名字（比如 `make_hongshao_rou`）。以后，每当你想做这道菜时，只需要“调用”这个菜谱（`make_hongshao_rou()`），电脑就会自动执行里面所有预设好的步骤，而不需要你每次都重新写一遍。

简单来说，**函数** 是一段具有特定功能的、可重复使用的代码块。**定义函数** 就是创建这个“菜谱”，而**调用函数** 就是按照“菜谱”去执行任务。

### 2. 关键定义

-   **函数 (Function)**：一个独立的、完成特定功能的代码块。它可以被重复使用，提高代码的**可读性**、**可维护性**和**复用性**。
-   **定义函数 (Define a Function)**：使用 `def` 关键字创建一个新函数的过程。定义时，需要指定函数名、参数（可选）和函数体（要执行的代码）。
    -   **语法结构**：
        ```python
        def 函数名(参数列表):
            """函数文档字符串（可选，用于说明函数功能）"""
            # 函数体：要执行的代码
            return 返回值  # 可选，用于将结果传回给调用者
        ```
-   **调用函数 (Call a Function)**：通过函数名和括号 `()` 来执行函数中定义的代码。如果函数有参数，需要在括号内传入对应的值。
    -   **语法结构**：`函数名(实际参数)`
-   **参数 (Parameter)**：定义函数时，在括号内声明的变量，用于接收调用者传入的数据。
-   **返回值 (Return Value)**：函数执行完毕后，通过 `return` 语句返回给调用者的结果。如果没有 `return`，函数默认返回 `None`。

### 3. 示例与应用

**生活例子 1：计算购物总价**

假设你经常去超市购物，需要计算几件商品的总价。

-   **定义函数**：我们定义一个名为 `calculate_total` 的函数，它接收两个参数：`price`（单价）和 `quantity`（数量），然后计算并返回总价。
    ```python
    def calculate_total(price, quantity):
        """计算商品总价"""
        total = price * quantity
        return total
    ```

-   **调用函数**：当你买了3斤苹果（每斤5元）和2斤香蕉（每斤4元）时，可以这样调用函数：
    ```python
    apple_total = calculate_total(5, 3)   # 调用函数，传入单价5和数量3
    banana_total = calculate_total(4, 2)  # 调用函数，传入单价4和数量2
    print(f"苹果总价：{apple_total}元")   # 输出：苹果总价：15元
    print(f"香蕉总价：{banana_total}元")  # 输出：香蕉总价：8元
    ```

**生活例子 2：发送问候消息**

你想给不同的朋友发送个性化的问候。

-   **定义函数**：定义一个名为 `greet` 的函数，它接收一个参数 `name`（朋友的名字），然后打印一句问候语。
    ```python
    def greet(name):
        """向指定名字的朋友问好"""
        print(f"你好，{name}！今天过得怎么样？")
    ```

-   **调用函数**：分别给“小明”和“小红”发送问候。
    ```python
    greet("小明")  # 输出：你好，小明！今天过得怎么样？
    greet("小红")  # 输出：你好，小红！今天过得怎么样？
    ```

### 4. 常见误区

-   **误区一：忘记调用函数**
    -   **错误理解**：很多初学者认为定义了函数，代码就会自动执行。
    -   **正确理解**：定义函数只是“创建”了一个工具，它不会自动运行。你必须通过 `函数名()` 来“使用”它。例如，只写了 `def greet(name): ...`，程序不会打印任何东西，必须写 `greet("小明")` 才会执行。

-   **误区二：混淆参数与返回值**
    -   **错误理解**：认为函数内部修改了参数，外部变量也会跟着变（对于不可变类型如数字、字符串）。
    -   **正确理解**：函数通过**参数**接收数据，通过**返回值**输出结果。函数内部对参数（如数字、字符串）的修改，不会影响函数外部的原始变量。例如：
        ```python
        def add_one(x):
            x = x + 1
            return x

        num = 5
        result = add_one(num)
        print(num)    # 输出：5 （原始变量没变）
        print(result) # 输出：6 （返回值才是新的结果）
        ```

### 5. 思考题

假设你正在编写一个程序来管理一个图书馆的图书借阅。请思考：如何设计一个名为 `borrow_book` 的函数，使得它既能处理“成功借阅”的情况，也能处理“图书已被借出”或“读者不存在”等异常情况？这个函数应该接收哪些参数？又应该返回什么值，才能让调用者清晰地知道操作结果？
