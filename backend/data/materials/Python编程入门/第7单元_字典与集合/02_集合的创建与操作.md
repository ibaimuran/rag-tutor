## 第7单元 字典与集合

### 知识点：集合的创建与操作

#### 1. 核心概念

想象一下，你有一个装满各种颜色弹珠的袋子。这个袋子有几个特点：首先，里面**不会有两个完全相同的弹珠**，每种颜色最多只有一个；其次，你**不关心弹珠在袋子里的顺序**，你只关心“袋子里有什么颜色的弹珠”。在Python中，**集合（Set）** 就像这样一个神奇的“袋子”。

集合是一个**无序的、不重复的元素序列**。它的主要作用是进行**成员关系测试**（判断某个元素是否在集合中）和**去重**（从列表中删除重复项）。你可以对集合进行数学上的集合运算，比如并集、交集和差集。

#### 2. 关键定义

- **集合（Set）**：一种Python内置的数据结构，用于存储**无序**且**唯一**的元素。使用花括号 `{}` 或 `set()` 函数创建。
- **无序（Unordered）**：集合中的元素没有固定的索引位置，因此不能像列表那样通过下标访问。
- **唯一性（Uniqueness）**：集合会自动**去除重复**的元素。如果你试图添加一个已经存在的元素，集合不会有任何变化。
- **可变性（Mutable）**：集合本身是**可变**的，你可以向其中添加或删除元素。但集合中的元素必须是**不可变**的（例如数字、字符串、元组），不能是列表或字典。
- **集合运算（Set Operations）**：对两个或多个集合进行的数学运算，如**并集（Union）**、**交集（Intersection）**、**差集（Difference）** 和**对称差集（Symmetric Difference）**。

#### 3. 示例与应用

**生活例子1：统计班级里有多少种不同的姓氏**

假设你有一个包含全班同学姓氏的列表，你想知道班里一共有多少种不同的姓氏。

```python
# 原始列表，包含重复姓氏
surnames_list = ['张', '王', '李', '张', '赵', '王', '刘', '李']

# 使用集合去重
unique_surnames = set(surnames_list)

print(unique_surnames)  # 输出: {'刘', '张', '王', '赵', '李'} (顺序可能不同)
print(f"班里共有 {len(unique_surnames)} 种不同的姓氏。")  # 输出: 班里共有 5 种不同的姓氏。
```

**生活例子2：检查你的购物清单是否包含“禁买”物品**

你有一个购物清单，同时妈妈列了一个“今天不能买”的零食清单。你想快速知道你的清单里有没有“违禁品”。

```python
# 你的购物清单
my_list = {'苹果', '薯片', '牛奶', '巧克力'}
# 妈妈的禁买清单
forbidden_list = {'薯片', '可乐', '巧克力'}

# 使用交集运算，找出同时出现在两个清单中的物品
conflict_items = my_list & forbidden_list  # 等价于 my_list.intersection(forbidden_list)

if conflict_items:
    print(f"警告！以下物品在禁买清单中：{conflict_items}")
else:
    print("你的清单很安全，没有违禁品。")
# 输出: 警告！以下物品在禁买清单中：{'巧克力', '薯片'}
```

一个用户可能被多次打上“VIP”标签，使用集合可以确保用户标签的唯一性，避免重复。

#### 4. 常见误区

- **误区一：用 `{}` 创建空集合**
  在Python中，`my_set = {}` 创建的是一个**空字典（Dictionary）**，而不是空集合。要创建空集合，必须使用 `set()` 函数。

  ```python
  # 错误做法
  empty_wrong = {}
  print(type(empty_wrong))  # 输出: <class 'dict'>

  # 正确做法
  empty_correct = set()
  print(type(empty_correct))  # 输出: <class 'set'>
  ```

- **误区二：认为集合是有序的**
  初学者可能会认为集合的输出顺序与输入顺序一致。但集合是**无序**的，每次运行程序，集合元素的打印顺序都可能不同。因此，**不能依赖集合的顺序**进行任何操作。
