## 第2章 流程控制

### 循环语句

#### 1. 核心概念

想象一下，你需要向全班50个同学每人说一声“你好”。如果让你一个一个地说，你需要重复说50次“你好”。这很枯燥，而且容易数错。在编程中，我们也会遇到很多需要**重复执行**相同或类似操作的情况。

**循环语句**就是用来解决这个问题的。它允许我们编写一段代码，然后告诉计算机：“请重复执行这段代码指定的次数，或者直到某个条件不再满足为止”。计算机非常擅长做这种重复性的工作，而且永远不会觉得累或出错。

简单来说，循环就是让程序**“反复做同一件事”**，直到我们告诉它“停”。

#### 2. 关键定义

- **循环 (Loop)**：一种控制结构，允许重复执行一段代码块。
- **迭代 (Iteration)**：循环每执行一次，就称为一次**迭代**。
- **循环体 (Loop Body)**：被重复执行的代码块。
- **循环条件 (Loop Condition)**：一个布尔表达式，用于决定循环是否继续执行。只要条件为**真 (True)**，循环就会继续；条件为**假 (False)**，循环就会终止。
- **`for` 循环**：一种循环，通常用于**遍历**一个序列（如列表、字符串、范围）中的每一个元素。当你明确知道要循环多少次时，`for` 循环非常方便。
- **`while` 循环**：一种循环，只要指定的**循环条件**为真，就会一直重复执行。当你不知道具体要循环多少次，但知道何时应该停止时，`while` 循环非常有用。

#### 3. 示例与应用

**生活例子1：发通知（`for` 循环）**

老师要通知全班30位同学明天春游。老师可以创建一个名单，然后对名单上的**每一个**同学说：“明天春游，请准时到校。” 这就是一个典型的 `for` 循环：遍历名单中的每一个元素（同学），并执行相同的操作（发通知）。

```python
# Python 代码示例
students = ["小明", "小红", "小刚", "小丽"]  # 名单列表
for student in students:
    print(f"{student}，明天春游，请准时到校。")
```

**生活例子2：猜数字游戏（`while` 循环）**

你心里想一个1到100之间的数字，让朋友来猜。只要朋友**没有猜对**，游戏就**一直**进行，朋友需要继续猜。一旦朋友猜对了，循环条件（“没猜对”）变为假，游戏结束。这就是一个 `while` 循环：在条件（没猜对）为真的情况下，重复执行“猜数字”这个动作。

```python
# Python 代码示例（简化版）
secret_number = 42
guess = 0

while guess != secret_number:  # 只要猜的数字不等于秘密数字
    guess = int(input("请输入你猜的数字："))
    if guess < secret_number:
        print("猜小了！")
    elif guess > secret_number:
        print("猜大了！")

print("恭喜你，猜对了！")
```

**应用场景：计算总和**

计算从1加到100的总和。如果手动计算很麻烦，但使用循环可以轻松完成。

```python
total = 0
for i in range(1, 101):  # range(1, 101) 生成 1 到 100 的数字序列
    total = total + i
print(f"1到100的总和是：{total}")
```

#### 4. 常见误区

- **误区一：忘记更新循环变量，导致无限循环**
  在使用 `while` 循环时，循环体内必须包含能改变循环条件的语句。否则，条件永远为真，循环会无限执行下去，程序会“卡死”。

  **错误示例**：
  ```python
  count = 1
  while count <= 5:
      print("正在循环...")
      # 忘记写 count = count + 1，导致 count 永远是 1，永远小于等于 5
  ```
  这个程序会无限打印“正在循环...”。正确的做法是在循环体内加上 `count = count + 1`。

- **误区二：混淆 `for` 和 `while` 的适用场景**
  有些同学会试图用 `for` 循环去解决一个需要持续等待某个事件发生的场景（比如等待用户输入“quit”才退出），或者用 `while` 循环去遍历一个已知长度的列表。虽然技术上可能实现，但会让代码变得复杂难懂。记住：**明确知道循环次数用 `for`，不确定次数但知道终止条件用 `while`**。
