## 第2章 流程控制

### 2.1 条件语句

#### 1. 核心概念

想象一下，你每天的生活都充满了选择：“如果下雨，我就带伞；否则，我就不带。” 计算机程序也需要做类似的决定。**条件语句**就是让程序能够根据不同的情况，执行不同代码块的“大脑”。它赋予程序“思考”和“判断”的能力，而不是机械地从头执行到尾。

简单来说，条件语句就是告诉计算机：“**如果**某个条件成立，**就**执行A操作；**否则**，就执行B操作。” 这个“条件”通常是一个可以被判断为“真”或“假”的命题。

#### 2. 关键定义

- **条件语句 (Conditional Statement)**：一种编程结构，允许程序根据特定条件的真假来选择执行不同的代码路径。
- **`if` 语句**：最基本的条件语句。它检查一个条件，如果条件为**真 (True)**，则执行其后的代码块。
- **`else` 语句**：与 `if` 语句配合使用。当 `if` 的条件为**假 (False)** 时，执行 `else` 后面的代码块。
- **`elif` 语句**：是“else if”的缩写。用于检查多个条件。当 `if` 的条件为假时，程序会依次检查每个 `elif` 的条件，一旦找到为真的条件，就执行对应的代码块。
- **布尔表达式 (Boolean Expression)**：一个计算结果为**真 (True)** 或**假 (False)** 的表达式。例如：`5 > 3` 的结果是 `True`，`10 == 20` 的结果是 `False`。
- **代码块 (Code Block)**：在条件语句中，被缩进（通常是4个空格或一个Tab键）的一组代码。它们被视为一个整体，在条件满足时一起执行。

#### 3. 示例与应用

**生活例子1：判断天气**

```python
# 场景：根据天气决定是否带伞
is_raining = True  # 假设外面下雨了

if is_raining:
    print("今天下雨，记得带伞！")
else:
    print("天气不错，不用带伞。")
```
**输出：** `今天下雨，记得带伞！`

**生活例子2：成绩等级评定**

```python
# 场景：根据考试分数给出等级
score = 85

if score >= 90:
    grade = "优秀"
elif score >= 80:
    grade = "良好"
elif score >= 70:
    grade = "中等"
elif score >= 60:
    grade = "及格"
else:
    grade = "不及格"

print(f"你的成绩等级是：{grade}")
```
**输出：** `你的成绩等级是：良好`

**生活例子3：登录验证**

```python
# 场景：模拟一个简单的登录系统
input_username = "admin"
input_password = "123456"

correct_username = "admin"
correct_password = "123456"

if input_username == correct_username and input_password == correct_password:
    print("登录成功！欢迎回来。")
else:
    print("用户名或密码错误，请重试。")
```
**输出：** `登录成功！欢迎回来。`

#### 4. 常见误区

- **误区一：混淆 `=` 和 `==`**
  - **错误理解**：在条件判断中，使用一个等号 `=` 来比较两个值是否相等。
  - **正确理解**：`=` 是**赋值运算符**，用于给变量赋值，例如 `x = 5`。`==` 是**比较运算符**，用于判断两个值是否相等，例如 `if x == 5:`。在条件语句中，必须使用 `==` 进行比较。

- **误区二：忘记冒号 `:`**
  - **错误理解**：在 `if`、`elif`、`else` 语句的末尾忘记写冒号。
  - **正确理解**：Python 使用冒号 `:` 来标记条件语句的开始，并告诉解释器下一行缩进的代码块属于这个条件。忘记冒号会导致语法错误。
